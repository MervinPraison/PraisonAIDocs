# PraisonAI UI

PraisonAI UI is a powerful, modern chat interface designed for AI agent interactions. It provides a beautiful UI for interacting with PraisonAI agents with features like streaming responses, tool call visualization, and session management.

## Installation

```bash
pip install praisonai-ui
```

## Quick Start

### CLI Usage

Run a chainlit app:

```bash
praisonai run app.py
```

With watch mode for development:

```bash
praisonai run app.py -w
```

This starts the chat server at `http://localhost:8000`.

### With Custom Port

```bash
praisonai run app.py --port 3000
```

### Basic App

```python
import praisonai_ui as ui

@ui.on_chat_start
async def start():
    await ui.Message(content="Hello! I'm your AI assistant.").send()

@ui.on_message
async def main(message: ui.Message):
    await ui.Message(content=f"You said: {message.content}").send()
```

Run with:
```bash
praisonai run app.py
```

## Step Decorator

```python
import praisonai_ui as ui

@ui.on_message
async def main(message: ui.Message):
    async with ui.Step(name="Processing") as step:
        step.output = "Thinking..."
    
    await ui.Message(content="Done!").send()
```

## Chat Profiles

```python
import praisonai_ui as ui

@ui.set_chat_profiles
async def chat_profile():
    return [
        ui.ChatProfile(name="GPT-4", markdown_description="OpenAI GPT-4"),
        ui.ChatProfile(name="Claude", markdown_description="Anthropic Claude"),
    ]

@ui.on_message
async def main(message: ui.Message):
    profile = ui.user_session.get("chat_profile")
    await ui.Message(content=f"Using {profile}").send()
```

## UI Features

### Chain of Thought Visualization

The chat interface displays agent reasoning steps, showing how agents arrive at their responses.

### Tool Call Panel

When agents use tools, the UI displays:
- Tool name
- Input arguments
- Tool output/results
- Execution time

### Session History

- Automatic session persistence
- Resume previous conversations
- Export chat history

## Integration with PraisonAI Agents

PraisonAI Chat integrates seamlessly with the full PraisonAI agent framework:

```python
from praisonaiagents import Agent, Task, Agents

# Define agents with tools
from praisonaiagents.tools import duckduckgo_search

researcher = Agent(
    name="Researcher",
    role="Research specialist",
    tools=[duckduckgo_search]
)

writer = Agent(
    name="Writer",
    role="Content writer"
)

# Define tasks
research_task = Task(
    description="Research {topic}",
    agent=researcher,
    expected_output="Research findings"
)

write_task = Task(
    description="Write article based on research",
    agent=writer,
    expected_output="Article content"
)

# Create the agents system
agents = Agents(
    agents=[researcher, writer],
    tasks=[research_task, write_task]
)

# The chat UI will automatically detect and use these agents
```

## Upstream Updates

PraisonAI UI is based on [Chainlit](https://github.com/Chainlit/chainlit), an excellent open-source framework. We maintain compatibility with upstream updates.

To sync with upstream improvements:

```bash
cd PraisonAI-UI
make sync-upstream
```
