# Face Blur Example

Blur faces using PraisonAI.

## Prerequisites

```bash
pip install praisonai praisonai-tools
```

## CLI Example

```bash
praisonai recipe run ai-face-blur \
  --input '{"input": "your-input-here"}' \
  --json
```

## Python Example

```python
from praisonai.recipes import run_recipe

result = run_recipe(
    "ai-face-blur",
    input={
        "input": "your-input-here"
    }
)

print(f"Success: {result['ok']}")
print(f"Output: {result['output']}")
```

## Server Example

```bash
# Start server
praisonai recipe serve --port 8080

# Invoke
curl -X POST http://localhost:8080/v1/recipes/run \
  -H "Content-Type: application/json" \
  -d '{
    "recipe": "ai-face-blur",
    "input": {"input": "your-input-here"}
  }'
```

## Expected Output

```json
{
  "ok": true,
  "run_id": "run_abc123",
  "recipe": "ai-face-blur",
  "output": {},
  "artifacts": []
}
```
