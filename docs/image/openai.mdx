## Generate

```python
from praisonaiagents import ImageAgent

agent = ImageAgent(llm="openai/dall-e-3")
result = agent.generate("A sunset over mountains")
print(result.data[0].url)
```

## With Options

```python
result = agent.generate(
    "A futuristic city",
    size="1792x1024",
    quality="hd",
    style="vivid"
)
```
