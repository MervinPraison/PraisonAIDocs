## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  instructions: 'Complete tasks efficiently',
  execution: {
    timeout: 30000,    // 30 second timeout
    retries: 3,        // Retry up to 3 times
    parallel: true     // Allow parallel execution
  }
});
```

## Execution Options

| Option | Default | Description |
|--------|---------|-------------|
| `timeout` | `60000` | Max time in milliseconds |
| `retries` | `0` | Number of retry attempts |
| `parallel` | `false` | Run independent tasks in parallel |
| `maxConcurrent` | `5` | Max parallel tasks |

## Common Patterns

### With Retries

```typescript
const agent = new Agent({
  execution: {
    retries: 3,
    retryDelay: 1000  // 1 second between retries
  }
});
```

### Parallel Execution

```typescript
// Run multiple agents in parallel
const results = await Promise.all([
  agent1.chat('Task A'),
  agent2.chat('Task B'),
  agent3.chat('Task C')
]);
```
