# Run with custom session ID
SESSION_ID=my-session npx ts-node examples/sessions/basic-session.ts
```

# Run multi-run session example
npx ts-node examples/sessions/multi-run.ts
```

```bash
# Run trace tracking example
npx ts-node examples/sessions/trace-tracking.ts
```
