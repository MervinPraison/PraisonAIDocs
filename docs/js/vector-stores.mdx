# Vector Store Integrations

PraisonAI TypeScript provides native integrations with popular vector databases for semantic search and RAG applications.

## Supported Vector Stores

| Store | Description | Best For |
|-------|-------------|----------|
| **Pinecone** | Managed vector database | Production, scale |
| **Weaviate** | Open-source with GraphQL | Hybrid search |
| **Qdrant** | High-performance | Self-hosted |
| **Chroma** | Lightweight, embedded | Development |
| **Memory** | In-memory | Testing |

## Quick Start

```typescript
import { 
  createPineconeStore,
  createWeaviateStore,
  createQdrantStore,
  createChromaStore,
  createMemoryVectorStore
} from 'praisonai';

// Pinecone
const pinecone = createPineconeStore({
  apiKey: process.env.PINECONE_API_KEY!
});

// Weaviate
const weaviate = createWeaviateStore({
  host: 'your-cluster.weaviate.network',
  apiKey: process.env.WEAVIATE_API_KEY
});

// Qdrant
const qdrant = createQdrantStore({
  url: 'http://localhost:6333',
  apiKey: process.env.QDRANT_API_KEY
});

// Chroma
const chroma = createChromaStore({
  host: 'localhost',
  port: 8000
});

// In-memory (for testing)
const memory = createMemoryVectorStore();
```

## Common Operations

### Create Index

```typescript
await store.createIndex({
  indexName: 'documents',
  dimension: 1536,  // OpenAI embedding dimension
  metric: 'cosine'  // 'cosine' | 'euclidean' | 'dotProduct'
});
```

### Upsert Vectors

```typescript
await store.upsert({
  indexName: 'documents',
  vectors: [
    {
      id: 'doc-1',
      vector: [0.1, 0.2, ...],  // 1536 dimensions
      content: 'Document content here',
      metadata: { source: 'manual.pdf', page: 1 }
    },
    {
      id: 'doc-2',
      vector: [0.3, 0.4, ...],
      content: 'Another document',
      metadata: { source: 'guide.pdf', page: 5 }
    }
  ]
});
```

### Query Vectors

```typescript
const results = await store.query({
  indexName: 'documents',
  vector: queryEmbedding,
  topK: 5,
  filter: { source: 'manual.pdf' },
  includeMetadata: true
});

for (const result of results) {
  console.log(`ID: ${result.id}, Score: ${result.score}`);
  console.log(`Content: ${result.content}`);
}
```

### Delete Vectors

```typescript
// Delete by IDs
await store.delete({
  indexName: 'documents',
  ids: ['doc-1', 'doc-2']
});

// Delete by filter
await store.delete({
  indexName: 'documents',
  filter: { source: 'old-file.pdf' }
});
```

## Pinecone

```typescript
import { createPineconeStore } from 'praisonai';

const pinecone = createPineconeStore({
  apiKey: process.env.PINECONE_API_KEY!
});

// Create serverless index
await pinecone.createIndex({
  indexName: 'my-index',
  dimension: 1536,
  metric: 'cosine'
});

// List indexes
const indexes = await pinecone.listIndexes();

// Get index stats
const stats = await pinecone.describeIndex('my-index');
console.log(`Vectors: ${stats.count}, Dimension: ${stats.dimension}`);
```

## Weaviate

```typescript
import { createWeaviateStore } from 'praisonai';

const weaviate = createWeaviateStore({
  host: 'your-cluster.weaviate.network',
  apiKey: process.env.WEAVIATE_API_KEY,
  scheme: 'https'
});

// Create class (index)
await weaviate.createIndex({
  indexName: 'Document',
  dimension: 1536,
  metric: 'cosine'
});

// Query with GraphQL-style filtering
const results = await weaviate.query({
  indexName: 'Document',
  vector: embedding,
  topK: 10,
  filter: { category: 'technical' }
});
```

## Qdrant

```typescript
import { createQdrantStore } from 'praisonai';

const qdrant = createQdrantStore({
  url: 'http://localhost:6333',
  apiKey: process.env.QDRANT_API_KEY
});

await qdrant.createIndex({
  indexName: 'documents',
  dimension: 1536,

```

## With Knowledge Base

Combine vector stores with the Knowledge Base for RAG:

```typescript
import { KnowledgeBase, createPineconeStore } from 'praisonai';

const vectorStore = createPineconeStore({
  apiKey: process.env.PINECONE_API_KEY!
});

const kb = new KnowledgeBase({
  vectorStore,
  embedder: async (text) => {
    // Your embedding function
    return await getEmbedding(text);
  }
});

// Add documents
await kb.add({
  id: 'doc-1',
  content: 'Document content...',
  metadata: { source: 'manual.pdf' }
});

// Search
const results = await kb.search('How to configure?', { topK: 5 });
```

## CLI Usage

```bash
# List indexes
npx praisonai vector list --store pinecone

# Create index
npx praisonai vector create my-index --dimension 1536 --store pinecone

# Query
npx praisonai vector query "search query" --index my-index --store pinecone
```
