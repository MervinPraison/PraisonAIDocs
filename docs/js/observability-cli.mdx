# Observability CLI Commands

The `praisonai-ts` CLI provides commands for monitoring and tracing.

## View Observability Info

```bash
# Show observability information
praisonai-ts observability info

# List available providers
praisonai-ts observability providers --json
```

**Example Output:**
```json
{
  "success": true,
  "data": {
    "providers": [
      { "name": "ConsoleObservabilityProvider", "description": "Console-based logging" },
      { "name": "MemoryObservabilityProvider", "description": "In-memory trace storage" },
      { "name": "LangfuseObservabilityProvider", "description": "Langfuse integration" }
    ]
  }
}
```

## Available Providers

| Provider | Description |
|----------|-------------|
| `ConsoleObservabilityProvider` | Console-based logging |
| `MemoryObservabilityProvider` | In-memory trace storage |
| `LangfuseObservabilityProvider` | Langfuse integration |

## SDK Usage

For programmatic observability:

```typescript
import { createConsoleObservability, createLangfuseObservability } from 'praisonai';

// Console observability
const consoleObs = createConsoleObservability();

// Langfuse observability
const langfuseObs = createLangfuseObservability({
  publicKey: process.env.LANGFUSE_PUBLIC_KEY,
  secretKey: process.env.LANGFUSE_SECRET_KEY
});
```

For more details, see the [Observability SDK documentation](/docs/js/observability).
