# Providers CLI Commands

The `praisonai-ts` CLI provides commands for managing LLM providers.

## List Providers

```bash
# List available providers
praisonai-ts providers

# Get JSON output
praisonai-ts providers --json
```

**Example Output:**
```json
{
  "success": true,
  "data": {
    "providers": [
      { "name": "openai", "models": ["gpt-4o", "gpt-4o-mini", "gpt-4-turbo"] },
      { "name": "anthropic", "models": ["claude-3-opus", "claude-3-sonnet", "claude-3-haiku"] },
      { "name": "google", "models": ["gemini-pro", "gemini-2.0-flash"] }
    ]
  }
}
```

## Use Specific Provider

```bash
# Chat with OpenAI (default)
praisonai-ts chat "Hello"

# Chat with specific model
praisonai-ts chat "Hello" --model openai/gpt-4o

# Chat with Anthropic
praisonai-ts chat "Hello" --model anthropic/claude-3-sonnet

# Chat with Google
praisonai-ts chat "Hello" --model google/gemini-2.0-flash
```

## Environment Variables

```bash
# Set API keys
export OPENAI_API_KEY=your-key
export ANTHROPIC_API_KEY=your-key
export GOOGLE_API_KEY=your-key

# Set default model
export PRAISONAI_MODEL=openai/gpt-4o-mini
```

## SDK Usage

For programmatic provider usage:

```typescript
import { createProvider, getAvailableProviders } from 'praisonai';

// List providers
const providers = getAvailableProviders();

// Create specific provider
const openai = createProvider('openai/gpt-4o-mini');
const anthropic = createProvider('anthropic/claude-3-sonnet');
const google = createProvider('google/gemini-2.0-flash');
```

For more details, see the [Providers SDK documentation](/docs/js/providers).
