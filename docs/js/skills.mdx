PraisonAI TypeScript implements the Agent Skills standard for reusable agent capabilities.

## Installation

```bash
npm install praisonai
```

## Skill Manager

```typescript
import { SkillManager, createSkillManager } from 'praisonai';

const manager = createSkillManager({
  paths: ['./skills'],
  includeProject: true,
  includeUser: true
});

// Discover skills
const skills = await manager.discover();
console.log('Found skills:', skills.length);
```

## Loading Skills

```typescript
// Load from file
const skill = await manager.loadSkill('./my-skill/SKILL.md');

// Register manually
manager.register({
  metadata: {
    name: 'my-skill',
    description: 'A custom skill'
  },
  instructions: 'Skill instructions here...'
});
```

## Using Skills

```typescript
// Get skill by name
const skill = manager.get('my-skill');

// List all skills
const allSkills = manager.list();

// Generate prompt XML
const prompt = manager.generatePrompt(['skill1', 'skill2']);
```

## SKILL.md Format

```markdown
---
name: code-review
description: Review code for best practices and issues
license: MIT
compatibility: TypeScript, JavaScript
allowed-tools: Read Grep
metadata:
  author: praisonai
  version: "1.0"
---

# Code Review Skill

When reviewing code:
1. Check for security issues
2. Verify error handling
3. Ensure proper typing
4. Look for performance issues
```

## Parsing Skills

```typescript
import { parseSkillFile } from 'praisonai';

const content = fs.readFileSync('./SKILL.md', 'utf-8');
const skill = parseSkillFile(content);

console.log('Name:', skill.metadata.name);
console.log('Description:', skill.metadata.description);
console.log('Instructions:', skill.instructions);
```

## Validation

```typescript
const validation = manager.validate(skill);

if (!validation.valid) {
  console.log('Errors:', validation.errors);
}
```

## Discovery Paths

Skills are discovered from:
1. **Project**: `./.praison/skills/`, `./.claude/skills/`
2. **User**: `~/.praison/skills/`
3. **System**: `/etc/praison/skills/` (if enabled)

```typescript
const manager = createSkillManager({
  includeProject: true,  // Default: true
  includeUser: true,     // Default: true
  includeSystem: false,  // Default: false
  paths: ['./custom-skills']
});
```
