```bash
# Run guardrails example
npx ts-node examples/guardrails/basic.ts
```

```bash
# Run built-in guardrails example
npx ts-node examples/guardrails/builtin.ts
```

```bash
# Run guardrail manager example
npx ts-node examples/guardrails/manager.ts
```

```bash
# Test content against guardrails
npx ts-node -e "
const {builtinGuardrails} = require('praisonai');
const g = builtinGuardrails.maxLength(100);
g.run('Hello world').then(r => console.log(r.status));
"
```
