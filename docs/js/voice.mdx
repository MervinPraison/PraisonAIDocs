# Voice Integrations

PraisonAI TypeScript provides voice capabilities through OpenAI TTS/Whisper and ElevenLabs.

## Supported Providers

| Provider | TTS | STT | Voices |
|----------|-----|-----|--------|
| **OpenAI** | ✅ | ✅ (Whisper) | 6 |
| **ElevenLabs** | ✅ | ❌ | 100+ |

## Quick Start

```typescript
import { createOpenAIVoice, createElevenLabsVoice } from 'praisonai';

// OpenAI Voice (TTS + Whisper)
const openai = createOpenAIVoice({
  apiKey: process.env.OPENAI_API_KEY
});

// ElevenLabs Voice (TTS only)
const elevenlabs = createElevenLabsVoice({
  apiKey: process.env.ELEVENLABS_API_KEY
});
```

## Text-to-Speech

### OpenAI TTS

```typescript
import { createOpenAIVoice } from 'praisonai';
import { writeFileSync } from 'fs';

const voice = createOpenAIVoice();

// Generate speech
const audio = await voice.speak('Hello, how can I help you today?', {
  voice: 'nova',  // alloy, echo, fable, onyx, nova, shimmer
  speed: 1.0,
  format: 'mp3'
});

// Save to file
writeFileSync('output.mp3', audio);
```

### ElevenLabs TTS

```typescript
import { createElevenLabsVoice } from 'praisonai';

const voice = createElevenLabsVoice({
  apiKey: process.env.ELEVENLABS_API_KEY
});

// Get available voices
const speakers = await voice.getSpeakers();
console.log('Available voices:', speakers.map(s => s.name));

// Generate speech
const audio = await voice.speak('Welcome to PraisonAI!', {
  voice: speakers[0].id
});
```

## Speech-to-Text

### OpenAI Whisper

```typescript
import { createOpenAIVoice } from 'praisonai';
import { readFileSync } from 'fs';

const voice = createOpenAIVoice();

// Transcribe audio file
const audioBuffer = readFileSync('recording.mp3');
const transcript = await voice.listen(audioBuffer, {
  language: 'en',
  model: 'whisper-1'
});

console.log('Transcript:', transcript);
```

## Voice Agent Example

```typescript
import { Agent, createOpenAIVoice } from 'praisonai';
import { readFileSync, writeFileSync } from 'fs';

const voice = createOpenAIVoice();

const agent = new Agent({
  name: 'Voice Assistant',
  instructions: 'You are a helpful voice assistant.'
});

// Process voice input
async function processVoiceInput(audioPath: string) {
  // 1. Transcribe audio
  const audio = readFileSync(audioPath);
  const userMessage = await voice.listen(audio);
  console.log('User said:', userMessage);

  // 2. Get agent response
  const response = await agent.chat(userMessage);
  console.log('Agent response:', response);

  // 3. Convert to speech
  const responseAudio = await voice.speak(response, { voice: 'nova' });
  writeFileSync('response.mp3', responseAudio);
  
  return response;
}
```

## CLI Usage

```bash
# Text to speech
npx praisonai voice speak "Hello world" --voice nova --output hello.mp3

# Speech to text
npx praisonai voice listen recording.mp3

# List available voices
npx praisonai voice voices --provider openai
```
