# Deepgram Provider

Speech recognition with Deepgram.

## Environment Variables

```bash
export DEEPGRAM_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'DeepgramAgent',
  instructions: 'Process speech.',
  llm: 'deepgram/nova-2'
});
```
