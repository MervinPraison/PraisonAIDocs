# Sarvam Provider

## Environment Variables

```bash
export SARVAM_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'SarvamAgent',
  instructions: 'You are a helpful assistant.',
  llm: 'sarvam/model'
});

const response = await agent.chat('Hello!');
```
