# Luma Provider

Video generation with Luma AI.

## Environment Variables

```bash
export LUMA_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'LumaAgent',
  instructions: 'Generate videos.',
  llm: 'luma/dream-machine'
});
```
