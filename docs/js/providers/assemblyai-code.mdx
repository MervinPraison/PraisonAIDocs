# AssemblyAI Provider

Speech-to-text with AssemblyAI.

## Environment Variables

```bash
export ASSEMBLYAI_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'TranscriptionAgent',
  instructions: 'Transcribe audio.',
  llm: 'assemblyai/best'
});
```
