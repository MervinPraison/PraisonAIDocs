# Gladia Provider

Speech transcription with Gladia.

## Environment Variables

```bash
export GLADIA_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'GladiaAgent',
  instructions: 'Transcribe audio.',
  llm: 'gladia/whisper-large-v3'
});
```
