# Portkey Provider

AI Gateway with Portkey.

## Environment Variables

```bash
export PORTKEY_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'PortkeyAgent',
  instructions: 'You are a helpful assistant.',
  llm: 'portkey/openai/gpt-4o'
});
```
