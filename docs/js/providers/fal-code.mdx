# Fal Provider

Image generation with Fal.ai.

## Environment Variables

```bash
export FAL_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'FalAgent',
  instructions: 'Generate images.',
  llm: 'fal/flux-pro'
});
```
