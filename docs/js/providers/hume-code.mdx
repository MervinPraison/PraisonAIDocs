# Hume Provider

Emotion AI with Hume.

## Environment Variables

```bash
export HUME_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'HumeAgent',
  instructions: 'Analyze emotions.',
  llm: 'hume/evi'
});
```
