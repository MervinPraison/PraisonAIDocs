# SambaNova Provider

## Environment Variables

```bash
export SAMBANOVA_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'SambaNovaAgent',
  instructions: 'You are a helpful assistant.',
  llm: 'sambanova/model'
});

const response = await agent.chat('Hello!');
```
