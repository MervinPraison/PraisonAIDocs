# Rev.ai Provider

Speech transcription with Rev.ai.

## Environment Variables

```bash
export REVAI_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'RevAIAgent',
  instructions: 'Transcribe audio.',
  llm: 'revai/async'
});
```
