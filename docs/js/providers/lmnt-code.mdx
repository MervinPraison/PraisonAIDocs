# LMNT Provider

Text-to-speech with LMNT.

## Environment Variables

```bash
export LMNT_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'LMNTAgent',
  instructions: 'Convert text to speech.',
  llm: 'lmnt/aurora'
});
```
