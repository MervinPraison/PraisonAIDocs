# Qwen Provider

Alibaba Qwen models.

## Environment Variables

```bash
export DASHSCOPE_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'QwenAgent',
  instructions: 'You are a helpful assistant.',
  llm: 'qwen/qwen-turbo'
});
```
