# ElevenLabs Provider

Text-to-speech with ElevenLabs.

## Environment Variables

```bash
export ELEVENLABS_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'TTSAgent',
  instructions: 'Convert text to speech.',
  llm: 'elevenlabs/eleven_multilingual_v2'
});
```
