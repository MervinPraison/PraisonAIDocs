# Black Forest Labs Provider

FLUX image generation models.

## Environment Variables

```bash
export BFL_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'FluxAgent',
  instructions: 'Generate images.',
  llm: 'black-forest-labs/flux-pro'
});
```
