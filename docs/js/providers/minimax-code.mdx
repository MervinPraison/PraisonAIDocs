# MiniMax Provider

## Environment Variables

```bash
export MINIMAX_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'MiniMaxAgent',
  instructions: 'You are a helpful assistant.',
  llm: 'minimax/abab6-chat'
});
```
