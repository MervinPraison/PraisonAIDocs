# NVIDIA NIM Provider

## Environment Variables

```bash
export NVIDIA_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'NIMAgent',
  instructions: 'You are a helpful assistant.',
  llm: 'nvidia-nim/meta/llama3-70b-instruct'
});
```
