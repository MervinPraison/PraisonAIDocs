# Helicone Provider

Observability proxy with Helicone.

## Environment Variables

```bash
export HELICONE_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'HeliconeAgent',
  instructions: 'You are a helpful assistant.',
  llm: 'helicone/openai/gpt-4o'
});
```
