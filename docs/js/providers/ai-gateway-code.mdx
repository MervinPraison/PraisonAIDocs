# AI Gateway Provider

Unified AI gateway access.

## Environment Variables

```bash
export AI_GATEWAY_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'GatewayAgent',
  instructions: 'You are a helpful assistant.',
  llm: 'ai-gateway/model'
});
```
