# Zhipu AI Provider

## Environment Variables

```bash
export ZHIPU_API_KEY=...
```

## Quick Start

```typescript
import { Agent } from 'praisonai';

const agent = new Agent({
  name: 'ZhipuAgent',
  instructions: 'You are a helpful assistant.',
  llm: 'zhipu-ai/glm-4'
});
```
