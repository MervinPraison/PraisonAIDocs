## How to Create Templates Using CLI

<Steps>
  <Step title="Initialize New Template">
    ```bash
    praisonai templates init my-template
    ```
  </Step>
  
  <Step title="Navigate to Template">
    ```bash
    cd my-template
    ```
  </Step>
  
  <Step title="Customize Generated Files">
    Edit `TEMPLATE.yaml` and `agents.yaml` to match your requirements.
  </Step>
</Steps>

## How to Create Templates Manually

<Steps>
  <Step title="Create Directory Structure">
    ```bash
    mkdir my-template
    cd my-template
    touch TEMPLATE.yaml agents.yaml
    ```
  </Step>
  
  <Step title="Write TEMPLATE.yaml">
    ```yaml
    name: my-template
    version: "1.0.0"
    description: "My custom template"
    
    variables:
      task:
        description: "Task to perform"
        required: true
    
    requires:
      tools: [internet_search]
    ```
  </Step>
  
  <Step title="Write agents.yaml">
    ```yaml
    framework: praisonai
    topic: "{{task}}"
    
    roles:
      agent:
        role: Assistant
        goal: Complete the task
        tasks:
          main:
            description: "{{task}}"
    ```
  </Step>
</Steps>

## How to Create Templates from Existing Agents

<Steps>
  <Step title="Export Running Agent">
    ```python
    from praisonaiagents import Agent
    
    agent = Agent(
        name="researcher",
        role="Research Specialist",
        goal="Research topics thoroughly",
        tools=["internet_search"]
    )
    
    # Export to template format
    agent.export_template("./my-research-template")
    ```
  </Step>
  
  <Step title="Customize Exported Template">
    ```bash
    cd my-research-template
    # Edit TEMPLATE.yaml to add variables
    ```
  </Step>
</Steps>

## How to Create Templates from GitHub

<Steps>
  <Step title="Fork Repository">
    Fork the Agent-Recipes repository on GitHub.
  </Step>
  
  <Step title="Clone Your Fork">
    ```bash
    git clone https://github.com/YOUR_USERNAME/Agent-Recipes
    cd Agent-Recipes/agent_recipes/templates
    ```
  </Step>
  
  <Step title="Create New Template Directory">
    ```bash
    mkdir my-new-template
    # Copy structure from existing template
    cp -r ai-video-editor/* my-new-template/
    ```
  </Step>
  
  <Step title="Customize and Push">
    ```bash
    # Edit files
    git add .
    git commit -m "Add my-new-template"
    git push
    ```
  </Step>
</Steps>

## Template Creation Methods Comparison

| Method | Best For | Complexity |
|--------|----------|------------|
| CLI `init` | Quick start | Low |
| Manual | Full control | Medium |
| From Agent | Converting existing code | Medium |
| Clone | Modifying existing templates | Low |
| GitHub Fork | Contributing to community | Medium |
