## Quick Reference

| Task | Command |
|------|---------|
| List templates | `praisonai templates list` |
| Add from GitHub | `praisonai templates add github:user/repo/template` |
| Add local | `praisonai templates add ./my-template` |
| Add source | `praisonai templates add-sources github:user/repo` |
| Run template | `praisonai templates run <name>` |
| Get info | `praisonai templates info <name>` |
| Validate | `praisonai templates validate <path>` |
| Init new | `praisonai templates init <name>` |
