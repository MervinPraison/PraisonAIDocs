## How to Update a Template

<Steps>
  <Step title="Check Current Version">
    ```bash
    praisonai templates info my-template
    ```
  </Step>
  
  <Step title="Edit TEMPLATE.yaml">
    Update the version and make changes:
    
    ```yaml
    # TEMPLATE.yaml
    name: my-template
    version: "1.1.0"  # Increment version
    description: "Updated description"
    ```
  </Step>
  
  <Step title="Validate Changes">
    ```bash
    praisonai templates validate ./my-template
    ```
  </Step>
  
  <Step title="Test Updated Template">
    ```bash
    praisonai templates run ./my-template --task "Test task"
    ```
  </Step>
</Steps>

## How to Edit Template Variables

<Steps>
  <Step title="View Current Variables">
    ```bash
    praisonai templates info my-template --show-variables
    ```
  </Step>
  
  <Step title="Add New Variables">
    ```yaml
    # TEMPLATE.yaml
    variables:
      task:
        description: "Main task"
        required: true
      output_format:
        description: "Output format"
        default: "markdown"
      verbose:
        description: "Enable verbose output"
        default: false
        type: boolean
    ```
  </Step>
  
  <Step title="Use Variables in agents.yaml">
    ```yaml
    roles:
      agent:
        tasks:
          main_task:
            description: |
              Task: {{task}}
              Format: {{output_format}}
              Verbose: {{verbose}}
    ```
  </Step>
</Steps>

## How to Delete a Template

<Steps>
  <Step title="List Installed Templates">
    ```bash
    praisonai templates list --installed
    ```
  </Step>
  
  <Step title="Remove Template">
    ```bash
    praisonai templates remove my-template
    ```
  </Step>
  
  <Step title="Verify Removal">
    ```bash
    praisonai templates list
    ```
  </Step>
</Steps>

## How to Clone and Modify a Template

<Steps>
  <Step title="Clone Existing Template">
    ```bash
    praisonai templates clone ai-video-editor my-video-editor
    ```
  </Step>
  
  <Step title="Modify Cloned Template">
    ```bash
    cd my-video-editor
    # Edit TEMPLATE.yaml, agents.yaml as needed
    ```
  </Step>
  
  <Step title="Run Modified Template">
    ```bash
    praisonai templates run ./my-video-editor --input video.mp4
    ```
  </Step>
</Steps>

## Template Management Commands

| Command | Description |
|---------|-------------|
| `praisonai templates list` | List all templates |
| `praisonai templates info <name>` | Show template details |
| `praisonai templates validate <path>` | Validate template |
| `praisonai templates clone <src> <dest>` | Clone template |
| `praisonai templates remove <name>` | Remove template |
