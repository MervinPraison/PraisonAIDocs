## How to Debug Template Execution

<Steps>
  <Step title="Enable Verbose Mode">
    ```bash
    praisonai templates run my-template --verbose
    ```
  </Step>
  
  <Step title="Check Template Validation">
    ```bash
    praisonai templates validate ./my-template
    ```
  </Step>
  
  <Step title="View Template Info">
    ```bash
    praisonai templates info my-template --show-variables
    ```
  </Step>
  
  <Step title="Check Tool Resolution">
    ```bash
    praisonai tools show-sources --template my-template
    ```
  </Step>
</Steps>

## How to Use Templates Doctor

<Steps>
  <Step title="Run Doctor Command">
    ```bash
    praisonai tools doctor
    ```
  </Step>
  
  <Step title="Check Specific Template">
    ```bash
    praisonai tools doctor --template my-template
    ```
  </Step>
  
  <Step title="Review Diagnostics">
    The doctor command will show:
    - Missing tools
    - Invalid tool references
    - Configuration issues
    - Dependency problems
  </Step>
  
  <Step title="Fix Issues">
    Based on doctor output, fix the identified issues in your template files.
  </Step>
</Steps>

## How to Debug Tool Resolution

<Steps>
  <Step title="List Available Tools">
    ```bash
    praisonai tools list
    ```
  </Step>
  
  <Step title="Resolve Specific Tool">
    ```bash
    praisonai tools resolve shell_tool
    ```
  </Step>
  
  <Step title="Check Tool Sources">
    ```bash
    praisonai tools show-sources --template my-template
    ```
  </Step>
  
  <Step title="Discover Tools from Packages">
    ```bash
    praisonai tools discover --include praisonai_tools.video
    ```
  </Step>
</Steps>

## How to Debug with Python

<Steps>
  <Step title="Enable Debug Logging">
    ```python
    import logging
    logging.basicConfig(level=logging.DEBUG)
    
    from praisonai.templates.loader import TemplateLoader
    loader = TemplateLoader()
    template = loader.load_template("my-template")
    ```
  </Step>
  
  <Step title="Inspect Template Config">
    ```python
    print(f"Name: {template.name}")
    print(f"Variables: {template.variables}")
    print(f"Requires: {template.requires}")
    ```
  </Step>
  
  <Step title="Check Tool Registry">
    ```python
    from praisonai.templates.tool_override import create_tool_registry_with_overrides
    
    registry = create_tool_registry_with_overrides(
        template_dir=template.path,
        include_defaults=True
    )
    print(f"Available tools: {list(registry.keys())}")
    ```
  </Step>
</Steps>

## Common Issues and Solutions

| Issue | Cause | Solution |
|-------|-------|----------|
| Tool not found | Tool not in registry | Add to `requires.tools` or `tools_sources` |
| Variable undefined | Missing variable | Add to `variables` in TEMPLATE.yaml |
| Template not found | Wrong path | Use full path or check template name |
| Permission denied | File access | Check file permissions |

## Debug CLI Options

```bash
praisonai templates run <template> [DEBUG OPTIONS]

Debug Options:
  --verbose              Enable verbose logging
  --dry-run              Show what would be executed
  --strict-tools         Fail on missing tools
```
