Ultra-fast transcription using Groq's Whisper implementation.

## Setup

```bash
export GROQ_API_KEY=your-key
```

## Usage

```python
from praisonaiagents import AudioAgent

agent = AudioAgent(llm="groq/whisper-large-v3")
text = agent.transcribe("audio.mp3")
print(text)
```

## Models

| Model | Description |
|-------|-------------|
| `groq/whisper-large-v3` | Best quality |
| `groq/whisper-large-v3-turbo` | Faster |
| `groq/distil-whisper-large-v3-en` | English only, fastest |

## With Language

```python
text = agent.transcribe("audio.mp3", language="es")
```

<Tip>
Groq offers the fastest Whisper inference available - often 10x faster than OpenAI.
</Tip>
