## Text-to-Speech

```python
from praisonaiagents import AudioAgent

agent = AudioAgent(llm="openai/tts-1")
agent.say("Hello!", output="hello.mp3")
```

## Speech-to-Text

```python
from praisonaiagents import AudioAgent

agent = AudioAgent(llm="openai/whisper-1")
text = agent.listen("audio.mp3")
print(text)
```
