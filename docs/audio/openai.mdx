Text-to-Speech and Speech-to-Text using OpenAI models.

## Setup

```bash
export OPENAI_API_KEY=your-key
```

## Text-to-Speech

```python
from praisonaiagents import AudioAgent

agent = AudioAgent(llm="openai/tts-1")
agent.speech("Hello world!", output="hello.mp3")
```

### TTS Models

| Model | Description |
|-------|-------------|
| `openai/tts-1` | Standard quality, fast |
| `openai/tts-1-hd` | High definition |
| `openai/gpt-4o-mini-tts` | GPT-4o mini TTS (newest) |

### Voices

| Voice | Description |
|-------|-------------|
| `alloy` | Neutral |
| `echo` | Male |
| `fable` | Expressive |
| `onyx` | Deep male |
| `nova` | Female |
| `shimmer` | Soft female |

```python
agent.speech("Hello!", voice="nova", output="hello.mp3")
```

## Speech-to-Text

```python
from praisonaiagents import AudioAgent

agent = AudioAgent(llm="openai/whisper-1")
text = agent.transcribe("audio.mp3")
print(text)
```

### STT Models

| Model | Description |
|-------|-------------|
| `openai/whisper-1` | Whisper large-v2 |

### With Language

```python
text = agent.transcribe("audio.mp3", language="en")
```
