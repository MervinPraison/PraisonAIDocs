Convert text to speech (TTS) and transcribe audio to text (STT) using AI models.

## Quick Start

<Tabs>
<Tab title="Text-to-Speech">
```python
from praisonaiagents import AudioAgent

agent = AudioAgent(llm="openai/tts-1")
agent.speech("Hello world!", output="hello.mp3")
```
</Tab>
<Tab title="Speech-to-Text">
```python
from praisonaiagents import AudioAgent

agent = AudioAgent(llm="openai/whisper-1")
text = agent.transcribe("audio.mp3")
print(text)
```
</Tab>
</Tabs>

## Methods

| Method | Description |
|--------|-------------|
| `speech(text)` | Convert text to audio |
| `transcribe(file)` | Convert audio to text |
| `say(text, output)` | Quick TTS + save |
| `listen(file)` | Quick STT |
