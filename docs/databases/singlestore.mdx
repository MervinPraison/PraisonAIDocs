# SingleStore

Distributed SQL database with real-time analytics.

## Setup

```bash
pip install singlestoredb
```

## Quick Start (Agent)

Use SingleStore for conversation persistence with an agent:

```python
from praisonaiagents import Agent, db

agent = Agent(
    name="Assistant",
    instructions="You are a helpful assistant.",
    db=db(database_url="singlestoredb://user:password@host:3306/database"),
    session_id="my-session"
)

agent.chat("Hello!")
```

## Advanced Usage (Direct Store)

```python
from praisonai.persistence.factory import create_conversation_store

store = create_conversation_store(
    "singlestore",
    url="singlestoredb://user:password@host:3306/database"
)
```
