# SurrealDB

Multi-model database for modern applications.

## Setup

```bash
docker run -d -p 8000:8000 surrealdb/surrealdb start
pip install surrealdb
```

## Quick Start (Agent)

Use SurrealDB for conversation persistence with an agent:

```python
from praisonaiagents import Agent, db

agent = Agent(
    name="Assistant",
    instructions="You are a helpful assistant.",
    db=db(database_url="surrealdb://localhost:8000/praisonai/conversations"),
    session_id="my-session"
)

agent.chat("Hello!")
```

## Advanced Usage (Direct Store)

```python
from praisonai.persistence.factory import create_conversation_store

store = create_conversation_store(
    "surrealdb",
    url="ws://localhost:8000/rpc",
    namespace="praisonai",
    database="conversations"
)
```
