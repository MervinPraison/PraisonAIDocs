# Redis CLI

## Docker Setup

```bash
docker run -d --name redis \
  -p 6379:6379 \
  redis:7
```

## Quick Start

```bash
# Test connection
praisonai persistence doctor \
  --state-backend redis \
  --state-url "redis://localhost:6379"

# Run with state persistence
praisonai persistence run \
  --state-backend redis \
  --state-url "$REDIS_URL" \
  "Hello"
```

## Commands

### Doctor
```bash
praisonai persistence doctor \
  --state-backend redis \
  --state-url "redis://localhost:6379"
```

### Run with State
```bash
praisonai persistence run \
  --state-backend redis \
  --state-url "$REDIS_URL" \
  --session-id my-session \
  "Process this task"
```
