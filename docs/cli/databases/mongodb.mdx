# MongoDB CLI

## Docker Setup

```bash
docker run -d --name mongodb \
  -p 27017:27017 \
  mongo:7
```

## Quick Start

```bash
# Test connection
praisonai persistence doctor \
  --state-backend mongodb \
  --state-url "mongodb://localhost:27017"

# Run with state
praisonai persistence run \
  --state-backend mongodb \
  --state-url "$MONGODB_URI" \
  "Hello"
```

## Commands

### Doctor
```bash
praisonai persistence doctor \
  --state-backend mongodb \
  --state-url "mongodb://localhost:27017"
```

### Run with State
```bash
praisonai persistence run \
  --state-backend mongodb \
  --state-url "$MONGODB_URI" \
  --session-id my-session \
  "Process this"
```
