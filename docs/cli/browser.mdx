Control browser automation for AI agents directly from the CLI. Open pages, take screenshots, click elements, and more.

## Commands

```bash
praisonai browser <command> [OPTIONS]
```

| Command | Description |
|---------|-------------|
| `status` | Check browser status |
| `open` | Open a URL in browser |
| `snapshot` | Get DOM snapshot |
| `screenshot` | Capture screenshot |
| `click` | Click an element |
| `type` | Type text into input |
| `navigate` | Navigate to URL |

---

## Status

Check if browser is running and get current page info:

```bash
praisonai browser status
```

**Output:**
```
Browser Status: Running
Current URL: https://example.com
Page Title: Example Domain
```

---

## Open URL

Open a URL in the browser:

```bash
praisonai browser open https://example.com
```

With options:
```bash
praisonai browser open https://example.com --headless --wait 5
```

| Option | Description |
|--------|-------------|
| `--headless` | Run in headless mode |
| `--wait SECONDS` | Wait after page load |

---

## Snapshot

Get a text snapshot of the current page DOM:

```bash
praisonai browser snapshot
```

Save to file:
```bash
praisonai browser snapshot --output page.txt
```

| Option | Description |
|--------|-------------|
| `--output FILE` | Save snapshot to file |
| `--selector CSS` | Snapshot specific element |

---

## Screenshot

Capture a screenshot of the current page:

```bash
praisonai browser screenshot
```

With options:
```bash
praisonai browser screenshot --output screenshot.png --full-page
```

| Option | Description |
|--------|-------------|
| `--output FILE` | Save screenshot to file (default: `screenshot.png`) |
| `--full-page` | Capture full scrollable page |
| `--selector CSS` | Screenshot specific element |

---

## Click

Click an element on the page:

```bash
praisonai browser click "#submit-button"
```

```bash
praisonai browser click "button.login" --wait 2
```

| Option | Description |
|--------|-------------|
| `--wait SECONDS` | Wait after click |
| `--force` | Force click even if hidden |

---

## Type

Type text into an input field:

```bash
praisonai browser type "#search-input" "AI agents"
```

```bash
praisonai browser type "input[name='email']" "user@example.com" --clear
```

| Option | Description |
|--------|-------------|
| `--clear` | Clear field before typing |
| `--delay MS` | Delay between keystrokes |
| `--submit` | Press Enter after typing |

---

## Navigate

Navigate to a URL (equivalent to `open` but for existing session):

```bash
praisonai browser navigate https://google.com
```

```bash
praisonai browser navigate /dashboard --wait 3
```

## Examples

### Web Scraping Workflow

```bash
# Open page
praisonai browser open https://news.ycombinator.com --headless

# Get page content
praisonai browser snapshot --output hn.txt

# Take screenshot for verification
praisonai browser screenshot --output hn.png
```

### Form Automation

```bash
# Navigate to login page
praisonai browser open https://example.com/login

# Fill form
praisonai browser type "#email" "user@example.com"
praisonai browser type "#password" "secret123"

# Submit
praisonai browser click "#login-button" --wait 3

# Verify success
praisonai browser screenshot --output logged-in.png
```

## Environment Variables

| Variable | Description |
|----------|-------------|
| `BROWSER_HEADLESS` | Run headless by default (`true`/`false`) |
| `BROWSER_TIMEOUT` | Default timeout in seconds |
