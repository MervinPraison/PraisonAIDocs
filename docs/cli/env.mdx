The `env` command displays environment and diagnostics information.

## Usage

```bash
praisonai env [OPTIONS] COMMAND [ARGS]...
```

## Commands

| Command | Description |
|---------|-------------|
| `show` | Show environment variables |
| `check` | Check environment configuration |

## Examples

### Show environment

```bash
praisonai env show
```

### Check configuration

```bash
praisonai env check
```
