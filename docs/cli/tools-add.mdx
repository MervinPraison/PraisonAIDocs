## How to Add Tools from Python Packages

<Steps>
  <Step title="Install the Package">
    ```bash
    pip install pandas
    ```
  </Step>
  
  <Step title="Add Package to Tools">
    ```bash
    praisonai tools add pandas
    ```
  </Step>
  
  <Step title="Verify Tools Added">
    ```bash
    praisonai tools show-sources
    ```
  </Step>
  
  <Step title="Use in Agent">
    ```python
    import pandas as pd
    from praisonaiagents import Agent
    
    def analyze_csv(data: str) -> str:
        """Analyze CSV data using pandas.
        
        Args:
            data: CSV-formatted string
            
        Returns:
            Analysis summary
        """
        from io import StringIO
        df = pd.read_csv(StringIO(data))
        return f"Rows: {len(df)}, Columns: {list(df.columns)}"
    
    agent = Agent(
        name="data_analyst",
        role="Data Analyst",
        tools=[analyze_csv]
    )
    
    result = agent.start("Analyze: name,age\\nAlice,30\\nBob,25")
    ```
  </Step>
</Steps>

## How to Add Tools from Local Files

<Steps>
  <Step title="Create Tools File">
    ```python
    # my_tools.py
    def my_custom_tool(query: str) -> str:
        """Process a query.
        
        Args:
            query: Input query
            
        Returns:
            Processed result
        """
        return f"Processed: {query}"
    ```
  </Step>
  
  <Step title="Add File to Tools">
    ```bash
    praisonai tools add ./my_tools.py
    ```
  </Step>
  
  <Step title="Verify Installation">
    The file is copied to `~/.praison/tools/my_tools.py`
  </Step>
</Steps>

## How to Add Tools from GitHub

<Steps>
  <Step title="Add from GitHub Repository">
    ```bash
    praisonai tools add github:MervinPraison/PraisonAI-tools/praisonai_tools
    ```
  </Step>
  
  <Step title="Verify Download">
    Tools are downloaded to `~/.praison/tools/`
  </Step>
</Steps>

## Configuration File

Tools sources are stored in `~/.praison/tools_sources.yaml`:

```yaml
sources:
- pandas
- numpy
- ./my_tools.py
```
