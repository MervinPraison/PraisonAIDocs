The `--init` flag initializes a new agents.yaml configuration file with optional topic.

## Quick Start

```bash
praisonai --init "Customer Support Bot"
```

## Usage

```bash
praisonai --init [topic] [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--init` | Initialize agents with optional topic |
| `--framework` | Framework to use (crewai, autogen, praisonai) |

## Examples

### Basic Initialization
```bash
praisonai --init
```

### With Topic
```bash
praisonai --init "E-commerce Product Recommendations"
```

### With Framework
```bash
praisonai --init "Data Pipeline" --framework praisonai
```

## Generated Files

Creates an `agents.yaml` file with:
- Default agent configuration
- Sample tasks
- Basic tool setup

## Next Steps

After initialization:
1. Edit `agents.yaml` to customize agents
2. Add tools as needed
3. Run with `praisonai agents.yaml`
