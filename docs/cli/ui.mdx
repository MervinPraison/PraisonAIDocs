The `ui` command manages the PraisonAI web user interface.

## Usage

```bash
praisonai ui [OPTIONS]
```

## Options

| Option | Short | Description | Default |
|--------|-------|-------------|---------|
| `--port` | `-p` | Port to run UI on | `8080` |
| `--host` | `-h` | Host to bind to | `127.0.0.1` |
| `--public` | | Make UI publicly accessible | `false` |

## Examples

### Start UI

```bash
praisonai ui
```

### Start on custom port

```bash
praisonai ui --port 3000
```

### Make publicly accessible

```bash
praisonai ui --public --host 0.0.0.0
```
