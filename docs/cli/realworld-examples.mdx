## Overview

This guide covers 10 real-world examples of using PraisonAI CLI for common coding tasks. Each example includes the CLI command, expected behavior, and tips for best results.

## Prerequisites

```bash
pip install praisonai
export OPENAI_API_KEY=your-key-here
```

## Examples

### 1. Generate Code from Specification

Write a new function based on a description.

```bash
praisonai chat -m gpt-4o-mini "Write a Python function celsius_to_fahrenheit that converts Celsius to Fahrenheit"
```

**Expected Output:** A complete Python function with the conversion formula `F = C * 9/5 + 32`.

### 2. Debug and Explain Errors

Get help understanding and fixing bugs.

```bash
praisonai chat -m gpt-4o-mini "What bug exists in a divide function that returns a/b? How would you fix it?"
```

**Expected Output:** Explanation of division by zero issue and suggested fix with error handling.

### 3. Generate Unit Tests

Create test cases for existing functions.

```bash
praisonai chat -m gpt-4o-mini "Write 3 pytest test cases for a shipping cost calculator function"
```

**Expected Output:** Complete pytest test functions with assertions.

### 4. Refactor Legacy Code

Modernize old code patterns.

```bash
praisonai chat -m gpt-4o-mini "How would you refactor a config parser function that splits strings by semicolons?"
```

**Expected Output:** Modern approach using proper parsing, error handling, and type hints.

### 5. Explain Code

Understand what existing code does.

```bash
praisonai chat -m gpt-4o-mini "Explain what a function called process_records that iterates over a list of dictionaries would do"
```

**Expected Output:** Clear explanation of the function's purpose and behavior.

### 6. Fix Failing Tests

Debug and fix code that causes test failures.

```bash
praisonai chat -m gpt-4o-mini "How do I fix a discount calculator to reject negative discount percentages?"
```

**Expected Output:** Validation logic with ValueError for invalid inputs.

### 7. Add CLI Commands

Extend command-line interfaces.

```bash
praisonai chat -m gpt-4o-mini "How do I add a version command to an argparse CLI in Python?"
```

**Expected Output:** Code showing `--version` flag implementation.

### 8. Search and Analyze Code

Find patterns in codebases.

```bash
praisonai chat -m gpt-4o-mini "What are common Python functions that format currency values?"
```

**Expected Output:** Examples of currency formatting approaches.

### 9. Continue Previous Session

Resume a conversation with context.

```bash
# First message
praisonai chat -m gpt-4o-mini "Explain Python decorators"

# Continue the conversation
praisonai chat -m gpt-4o-mini --continue "Give me an example"
```

**Expected Output:** Example that builds on the previous decorator explanation.

### 10. Analyze Attached Files

Process files as part of the prompt.

```bash
praisonai chat -m gpt-4o-mini --file README.md "Summarize this README file"
```

**Expected Output:** Summary of the file's contents.

## Session Management

### List Sessions

```bash
praisonai session list
```

### Export Session

```bash
praisonai session export <session_id> --output session.json
```

### Import Session

```bash
praisonai session import session.json
```

## Tips for Best Results

1. **Be Specific:** Provide clear, detailed prompts
2. **Use Cheap Models:** Start with `gpt-4o-mini` for cost efficiency
3. **Continue Sessions:** Use `--continue` to maintain context
4. **Attach Files:** Use `--file` to provide code context
5. **Check Output:** Verify generated code before using

## Troubleshooting

### Command Not Found

Ensure PraisonAI is installed and in your PATH:

```bash
pip install praisonai
which praisonai
```

### API Key Issues

Set your API key:

```bash
export OPENAI_API_KEY=your-key-here
```

### Session Not Found

List available sessions:

```bash
praisonai session list
```
