## Usage

```python
from praisonaiagents import VideoAgent

agent = VideoAgent(llm="openai/sora-2")
video = agent.generate("A sunset timelapse")
print(video.id)
```

## With Options

```python
video = agent.generate(
    prompt="Ocean waves",
    seconds="8",
    size="1920x1080"
)
```
