Generate high-quality videos using OpenAI's Sora-2 model.

## Setup

```bash
export OPENAI_API_KEY=your-key
```

<Note>
Requires [organization verification](https://platform.openai.com/settings/organization/general) to access Sora.
</Note>

## Usage

```python
from praisonaiagents import VideoAgent

agent = VideoAgent(llm="openai/sora-2")
video = agent.start("A dog running on a beach", output="dog.mp4")
```

## Models

| Model | Description |
|-------|-------------|
| `openai/sora-2` | Standard quality |
| `openai/sora-2-pro` | Higher quality |

## Parameters

```python
video = agent.generate(
    prompt="Your description",
    seconds="4",           # 4, 8, or 12
    size="1280x720"        # 1280x720 or 720x1280
)
```

## Duration Options

- `"4"` - 4 seconds
- `"8"` - 8 seconds  
- `"12"` - 12 seconds
