## Available Endpoints

| Protocol | Endpoints | Description |
|----------|-----------|-------------|
| A2A | `GET /.well-known/agent.json`, `GET /status`, `POST /a2a` | Agent-to-Agent communication |
| AG-UI | `POST /agui`, `GET /status` | Frontend integration with CopilotKit |
