Video analysis agent using vision models for content understanding.

**Agents: 1** — Single agent with vision capabilities analyzes video content.

## Workflow

1. Receive video file
2. Process frames with vision model
3. Detect objects and events
4. Generate comprehensive analysis

## Setup

```bash
pip install praisonaiagents praisonai
export OPENAI_API_KEY="your-key"
```

## Run — Python

```python
from praisonaiagents import Agent, Task, PraisonAIAgents

agent = Agent(
    name="VideoAnalyst",
    instructions="You are a video analysis expert. Describe video content in detail.",
    llm="gpt-4o-mini"
)

task = Task(
    description="Analyze this video and summarize key events",
    expected_output="Comprehensive video analysis",
    agent=agent,
    images=["video.mp4"]
)

agents = PraisonAIAgents(agents=[agent], tasks=[task])
result = agents.start()
print(result)
```

## Run — CLI

```bash
# Analyze a video
praisonai "Summarize this video" --image video.mp4

# With verbose output
praisonai "Analyze video content" --image video.mp4 --verbose
```

## Run — agents.yaml

```yaml
framework: praisonai
topic: Video Analysis
roles:
  video_analyst:
    role: Video Analysis Specialist
    goal: Analyze videos and describe content
    backstory: You are an expert in video analysis
    llm: gpt-4o-mini
    tasks:
      analyze:
        description: Analyze this video and summarize key events
        expected_output: Comprehensive video analysis
        images:
          - video.mp4
```

```bash
praisonai agents.yaml
```

## Serve API

```python
from praisonaiagents import Agent

agent = Agent(
    name="VideoAnalyst",
    instructions="You are a video analysis expert.",
    llm="gpt-4o-mini"
)

agent.launch(port=8080)
```
