OCR and text extraction agent using vision models.

**Agents: 1** — Single agent with vision capabilities extracts text from images.

## Workflow

1. Receive image with text
2. Process with vision model
3. Extract text content
4. Preserve layout and formatting

## Setup

```bash
pip install praisonaiagents praisonai
export OPENAI_API_KEY="your-key"
```

## Run — Python

```python
from praisonaiagents import Agent, Task, PraisonAIAgents

agent = Agent(
    name="OCRAgent",
    instructions="You are an OCR expert. Extract all text from images preserving layout.",
    llm="gpt-4o-mini"
)

task = Task(
    description="Extract all text from this document image",
    expected_output="Extracted text with layout preserved",
    agent=agent,
    images=["document.jpg"]
)

agents = PraisonAIAgents(agents=[agent], tasks=[task])
result = agents.start()
print(result)
```

## Run — CLI

```bash
# Extract text from image
praisonai "Extract text from this document" --image document.jpg

# Describe image content
praisonai --image-describe path/to/image.jpg
```

## Run — agents.yaml

```yaml
framework: praisonai
topic: Text Extraction
roles:
  ocr_agent:
    role: OCR Specialist
    goal: Extract text from images
    backstory: You are an expert in text extraction
    llm: gpt-4o-mini
    tasks:
      extract:
        description: Extract all text from this document image
        expected_output: Extracted text with layout preserved
        images:
          - document.jpg
```

```bash
praisonai agents.yaml
```

## Serve API

```python
from praisonaiagents import Agent

agent = Agent(
    name="OCRAgent",
    instructions="You are an OCR expert.",
    llm="gpt-4o-mini"
)

agent.launch(port=8080)
```
