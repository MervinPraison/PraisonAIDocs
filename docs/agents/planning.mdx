Travel planning agent with web search for finding flights, hotels, and creating itineraries.

**Agents: 1** — Single agent with search tool handles research and planning.

## Workflow

1. Receive travel request
2. Search for flights and hotels
3. Compare options
4. Generate detailed itinerary

## Setup

```bash
pip install praisonaiagents praisonai duckduckgo-search
export OPENAI_API_KEY="your-key"
```

## Run — Python

```python
from praisonaiagents import Agent
from praisonaiagents.tools import duckduckgo

agent = Agent(
    name="TravelPlanner",
    instructions="You are a travel planning agent. Search for options and create detailed itineraries.",
    tools=[duckduckgo]
)

result = agent.start("Plan a 3-day trip to Tokyo - find flights, hotels, and attractions")
print(result)
```

## Run — CLI

```bash
# Plan a trip
praisonai "Plan a weekend trip to Paris" --web-search

# With planning mode
praisonai "Plan a trip to London" --planning --web-search
```

## Run — agents.yaml

```yaml
framework: praisonai
topic: Travel Planning
roles:
  travel_planner:
    role: Travel Planning Specialist
    goal: Create comprehensive travel plans
    backstory: You are an expert travel planner
    tools:
      - duckduckgo
    tasks:
      plan_trip:
        description: Plan a 3-day trip to Tokyo - find flights, hotels, and attractions
        expected_output: A detailed itinerary with recommendations and estimated costs
```

```bash
praisonai agents.yaml
```

## Serve API

```python
from praisonaiagents import Agent
from praisonaiagents.tools import duckduckgo

agent = Agent(
    name="TravelPlanner",
    instructions="You are a travel planning agent.",
    tools=[duckduckgo]
)

agent.launch(port=8080)
```

```bash
# Test the API
curl -X POST http://localhost:8080/chat \
  -H "Content-Type: application/json" \
  -d '{"message": "Plan a weekend getaway to Barcelona"}'
```
