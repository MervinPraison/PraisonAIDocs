PraisonAI Claw gives you a **complete AI assistant** — a web dashboard with chat, agents, memory, knowledge, and bots for Telegram, Discord, and Slack — all from a single install.

## Quick Start

<Steps>

<Step title="Install">

```bash
pip install "praisonai[claw]"
```

</Step>

<Step title="Set your API key">

```bash
export OPENAI_API_KEY="your-api-key-here"
```

</Step>

<Step title="Create a chat.yaml file">

```yaml
name: My Assistant
instructions: You are a helpful AI assistant.
model: gpt-4o-mini
welcome: "👋 Hi! How can I help?"

starters:
  - label: "Explain a concept"
    message: "Explain the difference between machine learning and deep learning"
    icon: "🧠"
  - label: "Write code"
    message: "Show me how to make an HTTP request in Python"
    icon: "💻"

tools:
  - web_search
  - calculate

features: true
```

</Step>

<Step title="Run it">

```bash
aiui run chat.yaml
```

Open **http://localhost:8000** — your AI assistant is live.

</Step>

</Steps>

## YAML Configuration

The `chat.yaml` file controls your entire assistant. Here's every option:

```yaml
# Required
name: My Assistant
instructions: |
  You are a helpful assistant. Provide clear,
  well-structured answers with citations when possible.

# Optional — AI model
model: gpt-4o-mini

# Optional — welcome/goodbye messages
welcome: "👋 Hello! How can I help?"
goodbye: "👋 Thanks for chatting! See you next time."

# Optional — conversation starters shown in the UI
starters:
  - label: "Search the web"
    message: "Search for the latest news about AI agents"
    icon: "🔍"
  - label: "Calculate"
    message: "Calculate 42 * 37 + 15"
    icon: "🔢"

# Optional — selectable agent profiles
profiles:
  - name: Code Expert
    description: Code review, debugging, and generation
    icon: 💻
  - name: Writer
    description: Creative and technical writing
    icon: ✍️
  - name: Data Analyst
    description: Data analysis and visualization
    icon: 📊

# Optional — give the agent tools
tools:
  - web_search
  - calculate

# Optional — enable all platform features
# (memory, sessions, schedules, guardrails, skills, approvals, hooks, workflows)
features: true

# Optional — persist conversations to disk
datastore: json
```

### Run options

```bash
aiui run chat.yaml                           # default
aiui run chat.yaml --port 3000               # custom port
aiui run chat.yaml --style dashboard         # dashboard layout
aiui run chat.yaml --style agents            # tabbed multi-agent
aiui run chat.yaml --style playground        # input/output panels
aiui run chat.yaml --datastore json          # persist sessions
aiui run chat.yaml --datastore json:/tmp/db  # custom persistence path
aiui run chat.yaml --reload                  # auto-reload on changes
```

## Dashboard Mode

For a full dashboard with sidebar pages, create `app.py`:

```python
import os
import praisonaiui as aiui

aiui.set_style("dashboard")
aiui.set_pages([
    "chat",        # AI chat
    "channels",    # Messaging platforms
    "agents",      # Agent management
    "skills",      # Available tools
    "memory",      # Agent memory
    "knowledge",   # Knowledge base
    "sessions",    # Conversation history
    "guardrails",  # Safety rules
    "cron",        # Scheduled jobs
    "usage",       # Token usage & costs
    "config",      # Settings
])

aiui.register_agent({
    "agent_id": "researcher",
    "name": "Researcher",
    "description": "Finds and summarizes information",
    "instructions": "You are a research assistant.",
    "model": os.getenv("PRAISONAI_MODEL", "gpt-4o-mini"),
    "icon": "🔬",
})

aiui.register_agent({
    "agent_id": "coder",
    "name": "Coder",
    "description": "Writes clean, well-commented code",
    "instructions": "You are a coding assistant.",
    "model": os.getenv("PRAISONAI_MODEL", "gpt-4o-mini"),
    "icon": "💻",
})
```

```bash
aiui run app.py
```

## Connect Telegram

<Steps>

<Step title="Create a Telegram bot">

1. Open Telegram and search for **@BotFather**
2. Send `/newbot` and follow the prompts
3. Copy the **bot token** you receive

</Step>

<Step title="Set the token">

```bash
export TELEGRAM_BOT_TOKEN="123456:ABC-DEF..."
```

</Step>

<Step title="Create your bot">

Create `app.py`:

```python
import os
from praisonaiagents import Agent
from praisonai.bots.telegram import TelegramBot

agent = Agent(
    name="Assistant",
    instructions="You are a helpful assistant.",
    llm="gpt-4o-mini",
)

bot = TelegramBot(
    agent=agent,
    token=os.environ["TELEGRAM_BOT_TOKEN"],
)
bot.start()
```

</Step>

<Step title="Run it">

```bash
python app.py
```

Open your bot in Telegram and send a message — you'll get an AI response.

</Step>

</Steps>

## Connect Discord

<Steps>

<Step title="Create a Discord bot">

1. Go to the [Discord Developer Portal](https://discord.com/developers/applications)
2. Click **New Application** → name it → go to **Bot** → click **Reset Token**
3. Copy the **bot token**
4. Under **Privileged Gateway Intents**, enable **Message Content Intent**
5. Go to **OAuth2** → **URL Generator** → select `bot` scope → invite to your server

</Step>

<Step title="Set the token">

```bash
export DISCORD_BOT_TOKEN="MTIz..."
```

</Step>

<Step title="Create your bot">

Create `app.py`:

```python
import os
from praisonaiagents import Agent
from praisonai.bots.discord import DiscordBot

agent = Agent(
    name="Assistant",
    instructions="You are a helpful assistant.",
    llm="gpt-4o-mini",
)

bot = DiscordBot(
    agent=agent,
    token=os.environ["DISCORD_BOT_TOKEN"],
)
bot.start()
```

</Step>

<Step title="Run it">

```bash
python app.py
```

Mention your bot in Discord — it will respond with AI-generated messages.

</Step>

</Steps>

## Connect Slack

<Steps>

<Step title="Create a Slack app">

1. Go to [Slack API](https://api.slack.com/apps) → **Create New App** → **From Scratch**
2. Go to **OAuth & Permissions** → add scopes: `chat:write`, `app_mentions:read`, `im:history`, `im:read`
3. Click **Install to Workspace** and copy the **Bot User OAuth Token**

</Step>

<Step title="Set the token">

```bash
export SLACK_BOT_TOKEN="xoxb-..."
```

</Step>

<Step title="Create your bot">

Create `app.py`:

```python
import os
from praisonaiagents import Agent
from praisonai.bots.slack import SlackBot

agent = Agent(
    name="Assistant",
    instructions="You are a helpful assistant.",
    llm="gpt-4o-mini",
)

bot = SlackBot(
    agent=agent,
    token=os.environ["SLACK_BOT_TOKEN"],
)
bot.start()
```

</Step>

<Step title="Run it">

```bash
python app.py
```

Mention your bot in Slack — it will respond in the channel.

</Step>

</Steps>
