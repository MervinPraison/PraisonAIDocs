PraisonAI Claw gives you a **complete AI assistant** — a web dashboard with chat, agents, memory, knowledge, and bots for Telegram, Discord, and Slack — all from a single install.

## Getting Started

<Steps>

<Step title="Install">

```bash
pip install "praisonai[claw]"
```

</Step>

<Step title="Set your API key">

```bash
export OPENAI_API_KEY="your-api-key-here"
```

</Step>

<Step title="Launch the Dashboard">

Create `app.py`:

```python
import praisonaiui as aiui

aiui.set_style("dashboard")
aiui.set_pages([
    "chat",        # AI chat
    "channels",    # Messaging platforms
    "agents",      # Agent management
    "skills",      # Available tools
    "memory",      # Agent memory
    "knowledge",   # Knowledge base
    "sessions",    # Conversation history
    "config",      # Settings
])
```

Run it:

```bash
aiui run app.py
```

Open **http://localhost:8000** — your AI dashboard is live.

</Step>

</Steps>

## YAML Chat Mode

No Python needed — define your agent in a YAML file and run it directly:

```yaml
# chat.yaml
name: My Assistant
instructions: You are a helpful assistant.
model: gpt-4o-mini
welcome: "Hi! How can I help?"
starters:
  - label: "What can you do?"
    message: "Tell me about your capabilities"
tools:
  - web_search
  - calculate
features: true
```

```bash
aiui run chat.yaml
```

## Register Agents

Add AI agents to your dashboard:

```python
import os
import praisonaiui as aiui

aiui.set_style("dashboard")
aiui.set_pages(["chat", "agents", "memory", "knowledge", "skills"])
aiui.set_branding(title="PraisonAI", logo="🦞")

aiui.register_agent({
    "agent_id": "researcher",
    "name": "Researcher",
    "description": "Finds and summarizes information",
    "instructions": "You are a research assistant.",
    "model": os.getenv("PRAISONAI_MODEL", "gpt-4o-mini"),
    "icon": "🔬",
})

aiui.register_agent({
    "agent_id": "coder",
    "name": "Coder",
    "description": "Writes clean, well-commented code",
    "instructions": "You are a coding assistant.",
    "model": os.getenv("PRAISONAI_MODEL", "gpt-4o-mini"),
    "icon": "💻",
})
```

```bash
aiui run app.py
```

## Connect Telegram

<Steps>

<Step title="Create a Telegram bot">

1. Open Telegram and search for **@BotFather**
2. Send `/newbot` and follow the prompts
3. Copy the **bot token** you receive

</Step>

<Step title="Set the token">

```bash
export TELEGRAM_BOT_TOKEN="123456:ABC-DEF..."
```

</Step>

<Step title="Create your bot">

Create `app.py`:

```python
import os
from praisonaiagents import Agent
from praisonai.bots.telegram import TelegramBot

agent = Agent(
    name="Assistant",
    instructions="You are a helpful assistant.",
    llm="gpt-4o-mini",
)

bot = TelegramBot(
    agent=agent,
    token=os.environ["TELEGRAM_BOT_TOKEN"],
)
bot.start()
```

</Step>

<Step title="Run it">

```bash
python app.py
```

Open your bot in Telegram and send a message — you'll get an AI response.

</Step>

</Steps>

## Connect Discord

<Steps>

<Step title="Create a Discord bot">

1. Go to the [Discord Developer Portal](https://discord.com/developers/applications)
2. Click **New Application** → name it → go to **Bot** → click **Reset Token**
3. Copy the **bot token**
4. Under **Privileged Gateway Intents**, enable **Message Content Intent**
5. Go to **OAuth2** → **URL Generator** → select `bot` scope → invite to your server

</Step>

<Step title="Set the token">

```bash
export DISCORD_BOT_TOKEN="MTIz..."
```

</Step>

<Step title="Create your bot">

Create `app.py`:

```python
import os
from praisonaiagents import Agent
from praisonai.bots.discord import DiscordBot

agent = Agent(
    name="Assistant",
    instructions="You are a helpful assistant.",
    llm="gpt-4o-mini",
)

bot = DiscordBot(
    agent=agent,
    token=os.environ["DISCORD_BOT_TOKEN"],
)
bot.start()
```

</Step>

<Step title="Run it">

```bash
python app.py
```

Mention your bot in Discord — it will respond with AI-generated messages.

</Step>

</Steps>

## Connect Slack

<Steps>

<Step title="Create a Slack app">

1. Go to [Slack API](https://api.slack.com/apps) → **Create New App** → **From Scratch**
2. Go to **OAuth & Permissions** → add scopes: `chat:write`, `app_mentions:read`, `im:history`, `im:read`
3. Click **Install to Workspace** and copy the **Bot User OAuth Token**

</Step>

<Step title="Set the token">

```bash
export SLACK_BOT_TOKEN="xoxb-..."
```

</Step>

<Step title="Create your bot">

Create `app.py`:

```python
import os
from praisonaiagents import Agent
from praisonai.bots.slack import SlackBot

agent = Agent(
    name="Assistant",
    instructions="You are a helpful assistant.",
    llm="gpt-4o-mini",
)

bot = SlackBot(
    agent=agent,
    token=os.environ["SLACK_BOT_TOKEN"],
)
bot.start()
```

</Step>

<Step title="Run it">

```bash
python app.py
```

Mention your bot in Slack — it will respond in the channel.

</Step>

</Steps>
