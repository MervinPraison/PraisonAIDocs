PraisonAI Claw gives you a **complete AI assistant** — a web dashboard with chat, agents, memory, knowledge, and bots for Telegram, Discord, and Slack — all from a single install.

## Quick Start

<Steps>

<Step title="Install">

```bash
pip install "praisonai[claw]"
```

</Step>

<Step title="Set your API key">

```bash
export OPENAI_API_KEY="your-api-key-here"
```

</Step>

<Step title="Launch the dashboard">

```bash
praisonai claw
```

Open **http://localhost:8082** — the full dashboard is live with chat, agents, memory, knowledge, channels, and more.

</Step>

</Steps>

### CLI Options

```bash
praisonai claw                        # Dashboard on port 8082
praisonai claw --port 9000            # Custom port
praisonai claw --host 127.0.0.1      # Bind to localhost only
praisonai claw --app my-dashboard.py  # Custom app file
praisonai claw --reload               # Auto-reload on changes
```

### What happens under the hood

1. Checks that `praisonaiui` is installed (shows install instructions if missing)
2. Creates a default `app.py` at `~/.praisonai/claw/app.py` (if not already there)
3. Launches via `aiui run` — the real AIUI dashboard server

The default dashboard includes 13 pages and 3 built-in agents (Researcher, Writer, Coder).

## YAML Chat Mode

For a quick chat agent without any Python, create a `chat.yaml`:

```yaml
name: Research Assistant
instructions: |
  You are a helpful research assistant. Provide clear,
  well-structured answers with citations when possible.

model: gpt-4o-mini

welcome: "👋 Hello! I'm your Research Assistant. Ask me anything."

starters:
  - label: "Explain a concept"
    message: "Explain the difference between machine learning and deep learning"
    icon: "🧠"
  - label: "Write code"
    message: "Show me how to make an HTTP request in Python"
    icon: "💻"

tools:
  - web_search
  - calculate

features: true

datastore: json
```

```bash
aiui run chat.yaml
```

### YAML Configuration Reference

| Key | Type | Description |
|---|---|---|
| `name` | string | Agent display name |
| `instructions` | string | System prompt for the agent |
| `model` | string | LLM model (e.g., `gpt-4o-mini`) |
| `welcome` | string | Message shown when user connects |
| `goodbye` | string | Message shown when user disconnects |
| `starters` | list | Conversation starters with `label`, `message`, `icon` |
| `profiles` | list | Selectable agent personas with `name`, `description`, `icon` |
| `tools` | list | Built-in tools: `web_search`, `calculate`, etc. |
| `features` | boolean | Enable all 8 protocol features (memory, sessions, schedules, guardrails, skills, approvals, hooks, workflows) |
| `datastore` | string | Persistence: `json`, `memory`, or `json:/path/to/dir` |

### Run options

```bash
aiui run chat.yaml                           # default
aiui run chat.yaml --port 3000               # custom port
aiui run chat.yaml --style dashboard         # dashboard layout
aiui run chat.yaml --style agents            # tabbed multi-agent
aiui run chat.yaml --style playground        # input/output panels
aiui run chat.yaml --datastore json          # persist sessions
aiui run chat.yaml --datastore json:/tmp/db  # custom persistence path
aiui run chat.yaml --reload                  # auto-reload on changes
```

## Dashboard Mode

For a full dashboard with custom agents, create `app.py`:

```python
import os
import praisonaiui as aiui

aiui.set_style("dashboard")
aiui.set_pages([
    "chat",        # AI chat
    "channels",    # Messaging platforms
    "agents",      # Agent management
    "skills",      # Available tools
    "memory",      # Agent memory
    "knowledge",   # Knowledge base
    "sessions",    # Conversation history
    "guardrails",  # Safety rules
    "cron",        # Scheduled jobs
    "usage",       # Token usage & costs
    "config",      # Settings
])

aiui.register_agent({
    "agent_id": "researcher",
    "name": "Researcher",
    "description": "Finds and summarizes information",
    "instructions": "You are a research assistant.",
    "model": os.getenv("PRAISONAI_MODEL", "gpt-4o-mini"),
    "icon": "🔬",
})

aiui.register_agent({
    "agent_id": "coder",
    "name": "Coder",
    "description": "Writes clean, well-commented code",
    "instructions": "You are a coding assistant.",
    "model": os.getenv("PRAISONAI_MODEL", "gpt-4o-mini"),
    "icon": "💻",
})
```

```bash
aiui run app.py
```

## Docker

```bash
# Build
docker build -f docker/Dockerfile.claw -t praisonai:claw .

# Run dashboard
docker run -p 8082:8082 -e OPENAI_API_KEY=sk-xxx praisonai:claw

# Run bot instead
docker run -e OPENAI_API_KEY=sk-xxx -e TELEGRAM_BOT_TOKEN=... \
  praisonai:claw praisonai bot telegram

# Run gateway
docker run -e OPENAI_API_KEY=sk-xxx \
  praisonai:claw praisonai gateway start
```

## Connect Telegram

<Steps>

<Step title="Create a Telegram bot">

1. Open Telegram and search for **@BotFather**
2. Send `/newbot` and follow the prompts
3. Copy the **bot token** you receive

</Step>

<Step title="Set the token">

```bash
export TELEGRAM_BOT_TOKEN="123456:ABC-DEF..."
```

</Step>

<Step title="Create your bot">

Create `app.py`:

```python
import os
from praisonaiagents import Agent
from praisonai.bots.telegram import TelegramBot

agent = Agent(
    name="Assistant",
    instructions="You are a helpful assistant.",
    llm="gpt-4o-mini",
)

bot = TelegramBot(
    agent=agent,
    token=os.environ["TELEGRAM_BOT_TOKEN"],
)
bot.start()
```

</Step>

<Step title="Run it">

```bash
python app.py
```

Open your bot in Telegram and send a message — you'll get an AI response.

</Step>

</Steps>

## Connect Discord

<Steps>

<Step title="Create a Discord bot">

1. Go to the [Discord Developer Portal](https://discord.com/developers/applications)
2. Click **New Application** → name it → go to **Bot** → click **Reset Token**
3. Copy the **bot token**
4. Under **Privileged Gateway Intents**, enable **Message Content Intent**
5. Go to **OAuth2** → **URL Generator** → select `bot` scope → invite to your server

</Step>

<Step title="Set the token">

```bash
export DISCORD_BOT_TOKEN="MTIz..."
```

</Step>

<Step title="Create your bot">

Create `app.py`:

```python
import os
from praisonaiagents import Agent
from praisonai.bots.discord import DiscordBot

agent = Agent(
    name="Assistant",
    instructions="You are a helpful assistant.",
    llm="gpt-4o-mini",
)

bot = DiscordBot(
    agent=agent,
    token=os.environ["DISCORD_BOT_TOKEN"],
)
bot.start()
```

</Step>

<Step title="Run it">

```bash
python app.py
```

Mention your bot in Discord — it will respond with AI-generated messages.

</Step>

</Steps>

## Connect Slack

<Steps>

<Step title="Create a Slack app">

1. Go to [Slack API](https://api.slack.com/apps) → **Create New App** → **From Scratch**
2. Go to **OAuth & Permissions** → add scopes: `chat:write`, `app_mentions:read`, `im:history`, `im:read`
3. Click **Install to Workspace** and copy the **Bot User OAuth Token**

</Step>

<Step title="Set the token">

```bash
export SLACK_BOT_TOKEN="xoxb-..."
```

</Step>

<Step title="Create your bot">

Create `app.py`:

```python
import os
from praisonaiagents import Agent
from praisonai.bots.slack import SlackBot

agent = Agent(
    name="Assistant",
    instructions="You are a helpful assistant.",
    llm="gpt-4o-mini",
)

bot = SlackBot(
    agent=agent,
    token=os.environ["SLACK_BOT_TOKEN"],
)
bot.start()
```

</Step>

<Step title="Run it">

```bash
python app.py
```

Mention your bot in Slack — it will respond in the channel.

</Step>

</Steps>
