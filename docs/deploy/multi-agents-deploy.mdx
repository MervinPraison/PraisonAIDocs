Deploy multiple agents on one server using `Agents.launch()`.

## CLI

```bash
pip install praisonai
export OPENAI_API_KEY="your-key"

python -m praisonai --init "research and write content"
python -m praisonai --serve --port 8000
```

## Python

```python
from praisonaiagents import Agent, Agents

researcher = Agent(name="Researcher", instructions="Research topics", llm="gpt-4o-mini")
writer = Agent(name="Writer", instructions="Write content", llm="gpt-4o-mini")

agents = Agents(agents=[researcher, writer])
agents.launch(path="/content", port=8000)
```

## agents.yaml

```yaml
framework: praisonai
topic: research and write content
roles:
  researcher:
    role: Researcher
    goal: Research topics thoroughly
    backstory: Expert researcher
    tasks:
      research_task:
        description: Research the topic
        expected_output: Research findings
  writer:
    role: Writer
    goal: Write engaging content
    backstory: Expert writer
    tasks:
      write_task:
        description: Write based on research
        expected_output: Written content
```

```bash
python -m praisonai --serve
```

## Endpoints

| Endpoint | Description |
|----------|-------------|
| `POST /content` | Run full workflow |
| `POST /content/researcher` | Call researcher only |
| `POST /content/writer` | Call writer only |
| `GET /health` | Health check |
| `GET /docs` | Swagger UI |

## Verify

```bash
# Health check
curl http://localhost:8000/health

# Test workflow
curl -X POST http://localhost:8000/content \
  -H "Content-Type: application/json" \
  -d '{"query": "Write about AI"}'

# Test individual agent
curl -X POST http://localhost:8000/content/researcher \
  -H "Content-Type: application/json" \
  -d '{"query": "Research AI trends"}'
```

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Port in use | `lsof -i :8000` |
| Missing deps | `pip install "praisonaiagents[api]"` |
| No API key | `export OPENAI_API_KEY="your-key"` |
