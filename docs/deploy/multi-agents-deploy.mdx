Deploy multi-agent workflows using `Agents.launch()`.

## Prerequisites

```bash
pip install "praisonaiagents[api]"
export OPENAI_API_KEY="your-key"
```

## Quick Start

```python
from praisonaiagents import Agent, Agents

researcher = Agent(name="Researcher", instructions="Research topics thoroughly", llm="gpt-4o-mini")
writer = Agent(name="Writer", instructions="Write engaging content", llm="gpt-4o-mini")

agents = Agents(agents=[researcher, writer])
agents.launch(path="/content", port=8000)
```

## Test the API

```bash
# Workflow endpoint (all agents in sequence)
curl -X POST http://localhost:8000/content \
  -H "Content-Type: application/json" \
  -d '{"query": "Write about AI in healthcare"}'

# Individual agent endpoint
curl -X POST http://localhost:8000/content/Researcher \
  -H "Content-Type: application/json" \
  -d '{"query": "Research AI trends"}'
```

**Expected Response:**
```json
{
  "response": "# AI in Healthcare\n\nArtificial intelligence is transforming..."
}
```

## Agents.launch() Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `path` | str | `"/"` | Base API path |
| `port` | int | `8000` | Server port |
| `host` | str | `"0.0.0.0"` | Server host |
| `debug` | bool | `False` | Enable debug mode |
| `protocol` | str | `"http"` | `"http"` or `"mcp"` |

## Multiple Agent Groups

```python
from praisonaiagents import Agent, Agents

# Content team
content_team = Agents(agents=[
    Agent(name="Researcher", instructions="Research topics", llm="gpt-4o-mini"),
    Agent(name="Writer", instructions="Write content", llm="gpt-4o-mini")
])

# Code team
code_team = Agents(agents=[
    Agent(name="Analyzer", instructions="Analyze code", llm="gpt-4o-mini"),
    Agent(name="Reviewer", instructions="Review code", llm="gpt-4o-mini")
])

content_team.launch(path="/content", port=8000)
code_team.launch(path="/code", port=8000)
```

**Endpoints:**
- `POST /content` - Content workflow
- `POST /content/Researcher` - Researcher agent
- `POST /content/Writer` - Writer agent
- `POST /code` - Code workflow
- `POST /code/Analyzer` - Analyzer agent
- `POST /code/Reviewer` - Reviewer agent

## Docker Deployment

**app.py:**
```python
from praisonaiagents import Agent, Agents

agents = Agents(agents=[
    Agent(name="Assistant", instructions="Help users", llm="gpt-4o-mini")
])
agents.launch(path="/api", port=8000)
```

**Dockerfile:**
```dockerfile
FROM python:3.11-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt
COPY . .
EXPOSE 8000
CMD ["python", "app.py"]
```

**Build and Run:**
```bash
docker build -t multi-agent-api .
docker run -p 8000:8000 -e OPENAI_API_KEY=$OPENAI_API_KEY multi-agent-api
```

## Verify Deployment

```bash
# Health check
curl http://localhost:8000/health

# Expected response
{"status": "ok", "endpoints": ["/content", "/content/Researcher", "/content/Writer"]}
```

## Troubleshooting

| Issue | Check |
|-------|-------|
| Port in use | `lsof -i :8000` |
| Missing deps | `pip install "praisonaiagents[api]"` |
| No API key | `echo $OPENAI_API_KEY` |
| Agent not found | Check agent names match endpoint paths |
