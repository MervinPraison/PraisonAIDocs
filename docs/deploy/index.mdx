Deploy agents with one command.

## Quick Start

```bash
pip install "praisonaiagents[api]"
export OPENAI_API_KEY="your-key"
```

<Tabs>
  <Tab title="HTTP Server">
```python
from praisonaiagents import Agent

agent = Agent(instructions="You are helpful")
agent.launch(port=8000)
```
  </Tab>
  <Tab title="MCP Server">
```python
from praisonaiagents import Agent

agent = Agent(instructions="You are helpful")
agent.launch(port=8080, protocol="mcp")
```
  </Tab>
  <Tab title="CLI">
```bash
python -m praisonai --init "helpful assistant"
python -m praisonai --serve --port 8000
```
  </Tab>
</Tabs>

## Deploy as a Server

| Server Type | Use Case | Protocol |
|-------------|----------|----------|
| [Agents Server](./agents-server) | Single/Multi-agent HTTP API | HTTP REST |
| [MCP Server](./mcp-server) | Claude Desktop, Cursor integration | MCP |
| [A2A Server](./a2a-server) | Agent-to-Agent communication | A2A |
| [AGUI Server](./agui-server) | CopilotKit frontend integration | AG-UI |
| [Tools Server](./tools-server) | Expose tools as MCP server | MCP |
