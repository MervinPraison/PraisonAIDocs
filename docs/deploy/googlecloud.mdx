Deploy agents to Google Cloud Run.

## CLI

```bash
pip install praisonai
export OPENAI_API_KEY="your-key"

python -m praisonai --init "helpful assistant"
python -m praisonai --serve --port 8080 --host 0.0.0.0
```

## Python

**app.py:**
```python
from praisonaiagents import Agent

agent = Agent(instructions="You are a helpful assistant.", llm="gpt-4o-mini")
agent.launch(path="/ask", port=8080, host="0.0.0.0")
```

## Docker

**Dockerfile:**
```dockerfile
FROM python:3.11-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt
COPY app.py .
ENV PORT=8080
CMD ["python", "app.py"]
```

**requirements.txt:**
```
praisonaiagents[api]
```

**Build and test locally:**
```bash
docker build -t praisonai-agent .
docker run -p 8080:8080 -e OPENAI_API_KEY=$OPENAI_API_KEY praisonai-agent
```

## Verify

```bash
curl http://localhost:8080/health
curl -X POST http://localhost:8080/ask \
  -H "Content-Type: application/json" \
  -d '{"query": "Hello"}'
```

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Port in use | `lsof -i :8080` |
| Missing deps | `pip install "praisonaiagents[api]"` |
| No API key | `export OPENAI_API_KEY="your-key"` |

<Accordion title="Manual GCP CLI Validation (Optional)">

These commands are for validating your deployment on Google Cloud. Not required for PraisonAI deploy.

```bash
# Verify GCP CLI
gcloud config get-value project

# Push to Container Registry
gcloud auth configure-docker
docker build --platform linux/amd64 -t gcr.io/PROJECT_ID/praisonai-agent .
docker push gcr.io/PROJECT_ID/praisonai-agent

# Deploy to Cloud Run
gcloud run deploy praisonai-agent \
  --image gcr.io/PROJECT_ID/praisonai-agent \
  --platform managed \
  --region us-central1 \
  --allow-unauthenticated \
  --set-env-vars="OPENAI_API_KEY=$OPENAI_API_KEY"

# Verify deployment
gcloud run services describe praisonai-agent --format='value(status.url)'
```

</Accordion>
