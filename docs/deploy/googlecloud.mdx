Deploy agents to Google Cloud Run.

## Prerequisites

```bash
pip install "praisonaiagents[api]"
gcloud auth login
gcloud config set project YOUR_PROJECT_ID
```

## Verify GCP CLI

```bash
gcloud config get-value project
```

**Expected Output:**
```
your-project-id
```

## Cloud Run Deployment

### 1. Create Application

**app.py:**
```python
from praisonaiagents import Agent

agent = Agent(instructions="You are a helpful assistant.", llm="gpt-4o-mini")
agent.launch(path="/ask", port=8080, host="0.0.0.0")
```

**requirements.txt:**
```
praisonaiagents[api]
```

**Dockerfile:**
```dockerfile
FROM python:3.11-slim
WORKDIR /app
COPY requirements.txt .
RUN pip install --no-cache-dir -r requirements.txt
COPY . .
ENV PORT=8080
CMD ["python", "app.py"]
```

### 2. Build and Push

```bash
# Configure Docker
gcloud auth configure-docker

# Build and push
docker build -t gcr.io/PROJECT_ID/praisonai-agent .
docker push gcr.io/PROJECT_ID/praisonai-agent
```

### 3. Deploy

```bash
gcloud run deploy praisonai-agent \
  --image gcr.io/PROJECT_ID/praisonai-agent \
  --platform managed \
  --region us-central1 \
  --allow-unauthenticated \
  --set-env-vars="OPENAI_API_KEY=$OPENAI_API_KEY"
```

**Expected Output:**
```
Service [praisonai-agent] revision [praisonai-agent-00001-xxx] has been deployed
Service URL: https://praisonai-agent-xxxxx-uc.a.run.app
```

## Quick Deploy (All-in-One)

```bash
# Setup
gcloud init
gcloud services enable run.googleapis.com containerregistry.googleapis.com
gcloud auth configure-docker

# Build and deploy
PROJECT_ID=$(gcloud config get-value project)
docker build --platform linux/amd64 -t gcr.io/${PROJECT_ID}/praisonai-agent .
docker push gcr.io/${PROJECT_ID}/praisonai-agent

gcloud run deploy praisonai-agent \
  --image gcr.io/${PROJECT_ID}/praisonai-agent \
  --platform managed \
  --region us-central1 \
  --allow-unauthenticated \
  --set-env-vars="OPENAI_API_KEY=$OPENAI_API_KEY"
```

## Scaling

```bash
gcloud run services update praisonai-agent \
  --min-instances=1 \
  --max-instances=10 \
  --memory=2Gi \
  --cpu=1
```

## Secrets Management

```bash
# Create secret
echo -n "$OPENAI_API_KEY" | gcloud secrets create openai-api-key --data-file=-

# Deploy with secret
gcloud run deploy praisonai-agent \
  --image gcr.io/PROJECT_ID/praisonai-agent \
  --set-secrets=OPENAI_API_KEY=openai-api-key:latest
```

## Verify Deployment

```bash
# Get service URL
gcloud run services describe praisonai-agent --format='value(status.url)'

# Test endpoint
curl -X POST https://praisonai-agent-xxxxx-uc.a.run.app/ask \
  -H "Content-Type: application/json" \
  -d '{"query": "Hello"}'
```

## View Logs

```bash
gcloud logging read "resource.type=cloud_run_revision AND resource.labels.service_name=praisonai-agent" --limit=10
```

## Troubleshooting

| Issue | Check |
|-------|-------|
| Auth failed | `gcloud auth login` |
| No project | `gcloud config set project PROJECT_ID` |
| Push failed | `gcloud auth configure-docker` |
| Service not found | `gcloud run services list` |
