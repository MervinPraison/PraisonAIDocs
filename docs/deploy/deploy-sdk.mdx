Deploy agents programmatically using `Agent.launch()` or `Agents.launch()`.

## CLI

```bash
pip install praisonai
export OPENAI_API_KEY="your-key"

python -m praisonai --init "helpful assistant"
python -m praisonai --serve --port 8005
```

## Python - Single Agent

```python
from praisonaiagents import Agent

agent = Agent(instructions="You are a helpful assistant.", llm="gpt-4o-mini")
agent.launch(path="/ask", port=8000)
```

## Python - Multi-Agent

```python
from praisonaiagents import Agent, Agents

researcher = Agent(name="Researcher", instructions="Research topics", llm="gpt-4o-mini")
writer = Agent(name="Writer", instructions="Write content", llm="gpt-4o-mini")

agents = Agents(agents=[researcher, writer])
agents.launch(path="/content", port=8000)
```

## agents.yaml

```yaml
framework: praisonai
topic: helpful assistant
roles:
  assistant:
    role: Assistant
    goal: Help users
    backstory: You are helpful
    tasks:
      help_task:
        description: Answer questions
        expected_output: Helpful response
```

```bash
python -m praisonai --serve
```

## launch() Parameters

| Parameter | Type | Default | Description |
|-----------|------|---------|-------------|
| `path` | str | `"/"` | API endpoint path |
| `port` | int | `8000` | Server port |
| `host` | str | `"0.0.0.0"` | Server host |
| `debug` | bool | `False` | Debug mode |
| `protocol` | str | `"http"` | `"http"` or `"mcp"` |

## Verify

```bash
# Health check
curl http://localhost:8000/health

```

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Port in use | `lsof -i :8000` |
| Missing deps | `pip install "praisonaiagents[api]"` |
| No API key | `export OPENAI_API_KEY="your-key"` |
