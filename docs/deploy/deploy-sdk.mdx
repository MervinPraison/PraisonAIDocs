The `praisonai.deploy` module provides programmatic deployment for API, Docker, and Cloud targets.

## Prerequisites

```bash
pip install praisonai
```

## Deploy from YAML

```python
from praisonai.deploy import Deploy

deploy = Deploy.from_yaml("agents.yaml")
result = deploy.deploy()

print(f"Success: {result.success}")
print(f"URL: {result.url}")
```

**Expected Output:**
```
Success: True
URL: http://127.0.0.1:8005
```

## Deploy Types

### DeployType Enum

```python
from praisonai.deploy import DeployType

DeployType.API     # Local API server
DeployType.DOCKER  # Docker container
DeployType.CLOUD   # Cloud provider (AWS/Azure/GCP)
```

### API Server

```python
from praisonai.deploy import Deploy, DeployConfig, DeployType
from praisonai.deploy.models import APIConfig

config = DeployConfig(
    type=DeployType.API,
    api=APIConfig(
        host="0.0.0.0",
        port=8005,
        workers=1,
        cors_enabled=True,
        auth_enabled=False
    )
)

deploy = Deploy(config, "agents.yaml")
result = deploy.deploy()
```

### Docker

```python
from praisonai.deploy import Deploy, DeployConfig, DeployType
from praisonai.deploy.models import DockerConfig

config = DeployConfig(
    type=DeployType.DOCKER,
    docker=DockerConfig(
        image_name="my-agent",
        tag="latest",
        base_image="python:3.11-slim",
        expose=[8005],
        push=False
    )
)

deploy = Deploy(config, "agents.yaml")
result = deploy.deploy()
```

### Cloud (GCP)

```python
from praisonai.deploy import Deploy, DeployConfig, DeployType, CloudProvider
from praisonai.deploy.models import CloudConfig

config = DeployConfig(
    type=DeployType.CLOUD,
    cloud=CloudConfig(
        provider=CloudProvider.GCP,
        region="us-central1",
        service_name="my-agent-service",
        project_id="my-project-id",
        cpu="1",
        memory="512",
        min_instances=0,
        max_instances=10
    )
)

deploy = Deploy(config, "agents.yaml")
result = deploy.deploy()
```

## Doctor (Health Checks)

```python
from praisonai.deploy.doctor import run_local_checks, run_all_checks

# Local checks
report = run_local_checks()
for check in report.checks:
    status = "✅" if check.passed else "❌"
    print(f"{status} {check.name}: {check.message}")
```

**Expected Output:**
```
✅ Python Version: Python 3.12.11 (>= 3.9 required)
✅ Port 8005: Port 8005 is available
```

### Provider-Specific Checks

```python
from praisonai.deploy.doctor import (
    run_aws_checks,
    run_azure_checks,
    run_gcp_checks,
    run_all_checks
)

# Check all providers
report = run_all_checks()
print(f"Passed: {report.passed_checks}/{report.total_checks}")
```

**Expected Output:**
```
Passed: 6/6
```

## Status

```python
from praisonai.deploy import Deploy

deploy = Deploy.from_yaml("agents.yaml")
status = deploy.status()

print(f"State: {status.state.value}")
print(f"Healthy: {status.healthy}")
```

### ServiceState Values

```python
from praisonai.deploy import ServiceState

ServiceState.RUNNING    # Service is running
ServiceState.STOPPED    # Service is stopped
ServiceState.PENDING    # Service is starting
ServiceState.FAILED     # Service failed
ServiceState.NOT_FOUND  # Service doesn't exist
ServiceState.UNKNOWN    # Unknown state
```

## Destroy

```python
from praisonai.deploy import Deploy

deploy = Deploy.from_yaml("agents.yaml")
result = deploy.destroy(force=True)

print(f"Success: {result.success}")
print(f"Deleted: {result.resources_deleted}")
```

## YAML Configuration

### API Deploy

```yaml
name: My Agent
framework: praisonai

agents:
  assistant:
    name: Assistant
    role: Helper
    goal: Help users

deploy:
  type: api
  api:
    host: 0.0.0.0
    port: 8005
    workers: 1
    cors_enabled: true
```

### Docker Deploy

```yaml
deploy:
  type: docker
  docker:
    image_name: my-agent
    tag: latest
    base_image: python:3.11-slim
    expose:
      - 8005
```

### Cloud Deploy (GCP)

```yaml
deploy:
  type: cloud
  cloud:
    provider: gcp
    region: us-central1
    service_name: my-agent-service
    project_id: my-project-id
    cpu: "1"
    memory: "512"
```

## Schema Validation

```python
from praisonai.deploy.schema import validate_agents_yaml, generate_sample_yaml

# Validate
config = validate_agents_yaml("agents.yaml")

    print(f"Type: {config.type.value}")

# Generate sample
yaml_content = generate_sample_yaml(DeployType.API)

```

## Config Models

### APIConfig

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `host` | str | `"127.0.0.1"` | Server host |
| `port` | int | `8005` | Server port |
| `workers` | int | `1` | Worker processes |
| `cors_enabled` | bool | `True` | Enable CORS |
| `auth_enabled` | bool | `False` | Enable auth |
| `auth_token` | str | `None` | Auth token |

### DockerConfig

| Field | Type | Default | Description |
|-------|------|---------|-------------|
| `image_name` | str | `"praisonai-app"` | Image name |
| `tag` | str | `"latest"` | Image tag |
| `base_image` | str | `"python:3.11-slim"` | Base image |
| `expose` | list | `[8005]` | Ports to expose |
| `registry` | str | `None` | Registry URL |
| `push` | bool | `False` | Push after build |

### CloudConfig

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `provider` | CloudProvider | Yes | `aws`, `azure`, `gcp` |
| `region` | str | Yes | Deployment region |
| `service_name` | str | Yes | Service name |
| `project_id` | str | GCP only | GCP project ID |
| `resource_group` | str | Azure only | Azure resource group |
| `cpu` | str | No | CPU allocation |
| `memory` | str | No | Memory (MB) |
| `min_instances` | int | No | Min instances |
| `max_instances` | int | No | Max instances |

## Complete Example

```python
from praisonai.deploy import Deploy
from praisonai.deploy.doctor import run_local_checks

# 1. Health check
report = run_local_checks()
if not report.all_passed:
    for check in report.checks:
        if not check.passed:
            print(f"Fix: {check.fix_suggestion}")
    exit(1)

# 2. Deploy
deploy = Deploy.from_yaml("agents.yaml")
result = deploy.deploy()

if result.success:
    print(f"Deployed: {result.url}")
else:
    print(f"Failed: {result.error}")
```
