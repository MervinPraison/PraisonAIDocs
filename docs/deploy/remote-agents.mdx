Deploy agents accessible over the network using `host="0.0.0.0"`.

## CLI

```bash
pip install praisonai
export OPENAI_API_KEY="your-key"

python -m praisonai --init "helpful assistant"
python -m praisonai --serve --port 8000 --host 0.0.0.0
```

## Python

```python
from praisonaiagents import Agent

agent = Agent(instructions="You are a helpful assistant.", llm="gpt-4o-mini")
agent.launch(path="/ask", port=8000, host="0.0.0.0")
```

## agents.yaml

```yaml
framework: praisonai
topic: helpful assistant
roles:
  assistant:
    role: Assistant
    goal: Help users
    backstory: You are helpful
    tasks:
      help_task:
        description: Answer questions
        expected_output: Helpful response
```

```bash
python -m praisonai --serve --host 0.0.0.0
```

## Connect from Remote

```bash
curl -X POST http://SERVER_IP:8000/ask \
  -H "Content-Type: application/json" \
  -d '{"query": "Hello"}'
```

## Verify

```bash
# Health check
curl http://localhost:8000/health
```

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Connection refused | Use `host="0.0.0.0"` |
| Port in use | `lsof -i :8000` |
| Firewall blocking | Open port in firewall |
| No API key | `export OPENAI_API_KEY="your-key"` |
