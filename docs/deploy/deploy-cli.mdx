Deploy agents to Google Cloud using the `--deploy` flag.

## Prerequisites

```bash
pip install praisonai
gcloud auth login
gcloud config set project YOUR_PROJECT_ID
```

## Basic Deploy

```bash
praisonai --deploy
```

Deploys `agents.yaml` from current directory to Google Cloud Run.

**Expected Output:**
```
Starting deployment...
Creating API file...
Creating Dockerfile...
Building Docker image...
Pushing to Container Registry...
Deploying to Cloud Run...
Service URL: https://praisonai-service-xxxxx.run.app
```

## Deploy with Provider

```bash
praisonai --deploy --provider gcp
```

| Flag | Default | Options |
|------|---------|---------|
| `--provider` | `gcp` | `gcp`, `aws`, `azure` |

## Scheduled Deploy

```bash
praisonai --deploy --schedule "0 */6 * * *"
```

Deploys on a cron schedule (every 6 hours).

```bash
praisonai --deploy --schedule hourly --max-retries 5
```

| Flag | Description |
|------|-------------|
| `--schedule` | Cron expression or preset (`hourly`, `daily`) |
| `--max-retries` | Max retry attempts (default: 3) |
| `--schedule-config` | Path to schedule config YAML |

## Schedule Config File

```yaml
# deploy-config.yaml
deployment:
  schedule: "0 */6 * * *"
  provider: gcp
  max_retries: 5

environment:
  OPENAI_API_KEY: "${OPENAI_API_KEY}"
  MODEL_NAME: "gpt-4o-mini"
```

```bash
praisonai --deploy --schedule-config deploy-config.yaml
```

## Verify Deployment

```bash
# Check GCP project
gcloud config get-value project

# List Cloud Run services
gcloud run services list

# Get service URL
gcloud run services describe praisonai-service --format='value(status.url)'
```

## Troubleshooting

| Issue | Check |
|-------|-------|
| Auth failed | `gcloud auth login` |
| No project | `gcloud config set project PROJECT_ID` |
| Docker error | `docker info` |
| Push failed | `gcloud auth configure-docker` |

## Environment Variables

```bash
export OPENAI_API_KEY="your-key"
export OPENAI_MODEL_NAME="gpt-4o-mini"
export OPENAI_API_BASE="https://api.openai.com/v1"
```
