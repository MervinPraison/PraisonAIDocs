Deploy agents as an API server using `--serve`.

## CLI

```bash
pip install praisonai
export OPENAI_API_KEY="your-key"

# Create agents.yaml
python -m praisonai --init "helpful assistant"

# Start API server
python -m praisonai --serve
```

## CLI Flags

```bash
python -m praisonai --serve --port 8000 --host 0.0.0.0
```

| Flag | Default | Description |
|------|---------|-------------|
| `--serve` | - | Start API server |
| `--port` | `8005` | Server port |
| `--host` | `127.0.0.1` | Server host |

## Python

```python
from praisonaiagents import Agent

agent = Agent(instructions="You are a helpful assistant.", llm="gpt-4o-mini")
agent.launch(path="/ask", port=8000)
```

## agents.yaml

```yaml
framework: praisonai
topic: helpful assistant
roles:
  assistant:
    role: Assistant
    goal: Help users with their questions
    backstory: You are a helpful assistant
    tasks:
      help_task:
        description: Answer user questions
        expected_output: Helpful response
```

```bash
python -m praisonai --serve
```

## Verify

```bash
# Health check
curl http://localhost:8005/health

# Test endpoint
curl -X POST http://localhost:8005/agents \
  -H "Content-Type: application/json" \
  -d '{"query": "Hello"}'
```

## Troubleshooting

| Issue | Fix |
|-------|-----|
| Port in use | `lsof -i :8005` then kill process |
| No agents.yaml | `python -m praisonai --init "topic"` |
| Missing API key | `export OPENAI_API_KEY="your-key"` |
