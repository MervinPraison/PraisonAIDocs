## Quick Start

<Steps>
<Step title="Load Documents">
```rust
use praisonai::{Agent, Knowledge};

let knowledge = Knowledge::new()
    .add_file("report.pdf")
    .add_file("notes.txt")
    .add_directory("docs/")
    .build()?;

let agent = Agent::new()
    .name("Analyst")
    .knowledge(knowledge)
    .build()?;

agent.chat("Summarize the report").await?;
```
</Step>
</Steps>
