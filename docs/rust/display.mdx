## Quick Start

<Steps>
<Step title="Markdown Output">
```rust
use praisonai::Agent;

let agent = Agent::new()
    .name("Writer")
    .instructions("Format responses in markdown")
    .build()?;
```
</Step>

<Step title="Custom Formatting">
```rust
let agent = Agent::new()
    .name("Assistant")
    .on_message(|msg| {
        // Custom display logic
        render_markdown(&msg);
    })
    .build()?;
```
</Step>
</Steps>
