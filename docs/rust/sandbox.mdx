## Quick Start

<Steps>
<Step title="Enable Sandbox">
```rust
use praisonai::{Agent, SandboxConfig};

let config = SandboxConfig::new()
    .timeout_secs(30)
    .memory_limit_mb(256);

let agent = Agent::new()
    .name("Coder")
    .code_execution(true)
    .sandbox(config)
    .build()?;
```
</Step>
</Steps>

---

## Sandbox Options

| Option | Description |
|--------|-------------|
| `timeout_secs` | Execution timeout |
| `memory_limit_mb` | Memory limit |
| `allow_network` | Network access |
| `allow_fs` | File system access |
