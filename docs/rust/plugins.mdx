## Quick Start

<Steps>
<Step title="Add Plugin">
```rust
use praisonai::{Agent, Plugin};

let agent = Agent::new()
    .name("Assistant")
    .plugin(MyPlugin::new())
    .build()?;
```
</Step>
</Steps>

---

## Plugin Types

| Type | Purpose |
|------|---------|
| Tool plugins | Add new tools |
| Hook plugins | Intercept lifecycle |
| LLM plugins | Add providers |
