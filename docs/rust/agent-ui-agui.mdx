## Quick Start

<Steps>
<Step title="Web UI">
```rust
use praisonai::{Agent, WebUI};

let agent = Agent::new()
    .name("Assistant")
    .build()?;

// Start web interface
WebUI::new(agent)
    .port(8080)
    .start()
    .await?;

// Access at http://localhost:8080
```
</Step>
</Steps>
