## Quick Start

<Steps>
<Step title="Enable Code Execution">
```rust
use praisonai::Agent;

let agent = Agent::new()
    .name("Coder")
    .instructions("Write and execute code to solve problems")
    .code_execution(true)
    .build()?;

agent.chat("Calculate the first 10 Fibonacci numbers").await?;
// Agent writes Python code and runs it
```
</Step>
</Steps>

## Security

> [!CAUTION]
> Code execution allows running arbitrary code. Use in trusted environments only.

| Safety Feature | Description |
|----------------|-------------|
| Sandbox | Isolated execution environment |
| Timeout | Max execution time |
| Resource limits | Memory and CPU caps |
