## Quick Start

<Steps>
<Step title="Enable Thinking">
```rust
use praisonai::Agent;

let agent = Agent::new()
    .name("Thinker")
    .thinking(true)
    .build()?;

agent.chat("Solve this logic puzzle").await?;
// Shows step-by-step thinking
```
</Step>
</Steps>
