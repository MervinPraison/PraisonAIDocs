## Quick Start

<Steps>
<Step title="Apply Security">
```rust
use praisonai::{Agent, SecurityConfig};

let config = SecurityConfig::new()
    .allow_network(false)
    .allow_fs(false)
    .data_encryption(true);

let agent = Agent::new()
    .name("Secure Bot")
    .security(config)
    .build()?;
```
</Step>
</Steps>
