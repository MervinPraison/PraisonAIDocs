## Quick Start

<Steps>
<Step title="Enable Realtime">
```rust
use praisonai::{Agent, RealtimeConfig};

let config = RealtimeConfig::new()
    .websocket_enabled(true);

let agent = Agent::new()
    .name("Realtime Bot")
    .realtime(config)
    .build()?;

// Start WebSocket server
agent.serve(8080).await?;
```
</Step>
</Steps>
