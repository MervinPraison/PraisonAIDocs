## Quick Start

<Steps>

<Step title="Prerequisites">
```bash
# Install Rust (if not already installed)
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```
</Step>

<Step title="Add to Project">
```bash
cargo add praisonai
```
</Step>

<Step title="Set API Key">
```bash
export OPENAI_API_KEY="your-api-key"
```
</Step>

</Steps>

---

## Installation Methods

<Tabs>
<Tab title="Library">
Add to your `Cargo.toml`:

```toml
[dependencies]
praisonai = "0.1"
tokio = { version = "1", features = ["full"] }
anyhow = "1"
```

Or use cargo:
```bash
cargo add praisonai tokio anyhow
```
</Tab>

<Tab title="CLI">
```bash
# Install globally
cargo install praisonai-cli

# Verify installation
praisonai-rust --version
```
</Tab>
</Tabs>

## Environment Variables

| Variable | Required | Description |
|----------|----------|-------------|
| `OPENAI_API_KEY` | Yes | OpenAI API authentication |
| `PRAISONAI_MODEL` | No | Default model (gpt-4o-mini) |
| `RUST_LOG` | No | Enable debug logging |

---

## Verify Installation

```rust
use praisonai::Agent;

#[tokio::main]
async fn main() -> anyhow::Result<()> {
    let agent = Agent::simple("Say hello")?;
    let response = agent.chat("Hi!").await?;
    println!("{}", response);
    Ok(())
}
```

Run with:
```bash
cargo run
```
