## Quick Start

<Steps>
<Step title="Create Bot Agent">
```rust
use praisonai::Agent;

let agent = Agent::new()
    .name("Support Bot")
    .instructions("Help users with their questions")
    .build()?;

// Deploy to platform
bot::deploy(agent, "slack").await?;
```
</Step>
</Steps>
