## Quick Start

<Steps>
<Step title="Enable Self Reflection">
```rust
use praisonai::{Agent, ReflectionConfig};

let config = ReflectionConfig::new()
    .enabled()
    .criteria("Is this response helpful?");

let agent = Agent::new()
    .name("Writer")
    .reflection(config)
    .build()?;
```
</Step>
</Steps>
