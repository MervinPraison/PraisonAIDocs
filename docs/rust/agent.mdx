# Agent

Agents are the primary execution unit in PraisonAI. They combine an LLM provider, tools, memory, and instructions to create intelligent assistants.

## Quick Start

```rust
use praisonai::{Agent, Result};

#[tokio::main]
async fn main() -> Result<()> {
    let agent = Agent::new()
        .name("assistant")
        .instructions("You are a helpful assistant")
        .model("gpt-4o-mini")
        .build()?;
    
    let response = agent.chat("Hello!").await?;
    println!("{}", response);
    
    Ok(())
}
```

## Agent Builder

Create agents with the fluent builder pattern:

```rust
use praisonai::Agent;

let agent = Agent::new()
    .name("research-assistant")
    .instructions("You help with research tasks")
    .model("gpt-4o")
    .temperature(0.7)
    .max_tokens(2000)
    .max_iterations(15)
    .verbose(true)
    .stream(true)
    .build()?;
```

### Builder Methods

| Method | Description |
|--------|-------------|
| `name(str)` | Set agent name |
| `instructions(str)` | Set system instructions |
| `model(str)` | Set LLM model |
| `llm(str)` | Alias for `model()` |
| `api_key(str)` | Set API key |
| `base_url(str)` | Set custom API endpoint |
| `temperature(f32)` | Set temperature (0.0-2.0) |
| `max_tokens(u32)` | Set max response tokens |
| `max_iterations(usize)` | Set max tool call iterations |
| `verbose(bool)` | Enable verbose output |
| `stream(bool)` | Enable streaming |
| `tool(impl Tool)` | Add a single tool |
| `tools(iterator)` | Add multiple tools |
| `memory(bool)` | Enable/disable memory |
| `memory_config(MemoryConfig)` | Set memory configuration |

## Running Agents

Multiple methods for interacting with agents:

```rust
// Primary method
let response = agent.chat("What is Rust?").await?;

// Aliases
let response = agent.start("Begin the task").await?;
let response = agent.run("Execute this").await?;
```

## Simple Agent

Create a minimal agent quickly:

```rust
use praisonai::Agent;

// One-liner with just instructions
let agent = Agent::simple("You are a helpful assistant")?;
let response = agent.chat("Hello").await?;
```

## Adding Tools

Agents can use tools to perform actions:

```rust
use praisonai::{Agent, tool};
use serde_json::Value;

#[tool(description = "Search the web for information")]
async fn search_web(query: String) -> String {
    format!("Results for: {}", query)
}

let agent = Agent::new()
    .instructions("You can search the web")
    .tool(SearchWeb)
    .build()?;

let response = agent.chat("Search for Rust tutorials").await?;
```

### Multiple Tools

```rust
let agent = Agent::new()
    .instructions("You have multiple capabilities")
    .tool(SearchWeb)
    .tool(Calculator)
    .tool(FileReader)
    .build()?;

// Or with an iterator
let tools = vec![Box::new(SearchWeb), Box::new(Calculator)];
let agent = Agent::new()
    .tools(tools)
    .build()?;
```

### Dynamic Tool Addition

```rust
let agent = Agent::new().build()?;

// Add tools after creation
agent.add_tool(SearchWeb).await;
agent.add_tool(Calculator).await;

println!("Tools: {}", agent.tool_count().await);
```

## Memory

Agents maintain conversation history:

```rust
use praisonai::{Agent, MemoryConfig};

let agent = Agent::new()
    .memory_config(MemoryConfig::new().max_messages(50))
    .build()?;

// Multi-turn conversation
agent.chat("My name is Alice").await?;
agent.chat("What's my name?").await?;  // Remembers "Alice"

// Access history
let history = agent.history().await?;
for msg in history {
    println!("{}: {}", msg.role, msg.content);
}

// Clear memory
agent.clear_memory().await?;
```

## Agent Properties

```rust
let agent = Agent::new()
    .name("test-agent")
    .instructions("Be helpful")
    .model("gpt-4o")
    .build()?;

// Access properties
println!("ID: {}", agent.id());         // UUID
println!("Name: {}", agent.name());     // "test-agent"
println!("Model: {}", agent.model());   // "gpt-4o"
println!("Instructions: {}", agent.instructions());
```

## Configuration

Customize agent behavior:

```rust
use praisonai::{Agent, MemoryConfig, ExecutionConfig};

let agent = Agent::new()
    .name("configured-agent")
    .instructions("You are helpful")
    
    // LLM settings
    .model("gpt-4o")
    .temperature(0.8)
    .max_tokens(4096)
    .api_key("sk-...")
    .base_url("https://custom-endpoint.com/v1")
    
    // Execution settings
    .max_iterations(20)
    .verbose(true)
    .stream(false)
    
    // Memory settings
    .memory_config(MemoryConfig::new()
        .max_messages(100)
        .with_long_term())
    
    .build()?;
```

## Example: Research Agent

```rust
use praisonai::{Agent, tool, Result};

#[tool(description = "Search academic papers")]
async fn search_papers(query: String) -> String {
    // Search implementation
    format!("Found papers about: {}", query)
}

#[tool(description = "Summarize a paper")]
async fn summarize_paper(paper_id: String) -> String {
    format!("Summary of paper {}", paper_id)
}

#[tokio::main]
async fn main() -> Result<()> {
    let agent = Agent::new()
        .name("research-agent")
        .instructions(r#"
            You are a research assistant. You can:
            - Search for academic papers
            - Summarize papers
            Be thorough and cite sources.
        "#)
        .model("gpt-4o")
        .temperature(0.3)
        .tool(SearchPapers)
        .tool(SummarizePaper)
        .build()?;
    
    let result = agent.chat(
        "Find recent papers about large language models and summarize the top one"
    ).await?;
    
    println!("{}", result);
    Ok(())
}
```

## Default Agent

Create an agent with all defaults:

```rust
let agent = Agent::default();
assert!(!agent.id().is_empty());
assert_eq!(agent.name(), "agent");
```
