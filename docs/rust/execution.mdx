## Quick Start

<Steps>
<Step title="Set Execution Limits">
```rust
use praisonai::Agent;

let agent = Agent::new()
    .max_iterations(10)
    .timeout_seconds(120)
    .build()?;
```
</Step>

<Step title="With Configuration">
```rust
use praisonai::execution::ExecutionConfig;

let agent = Agent::new()
    .execution(ExecutionConfig::new()
        .max_iterations(5)
        .timeout_seconds(60)
        .retry_on_error(true)
        .max_retries(3))
    .build()?;
```
</Step>
</Steps>

---

## Configuration Options

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `max_iterations` | `usize` | `10` | Max agent loops |
| `timeout_seconds` | `u64` | `300` | Total timeout |
| `retry_on_error` | `bool` | `false` | Auto-retry on error |
| `max_retries` | `usize` | `3` | Max retry attempts |
