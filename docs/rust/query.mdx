## Quick Start

<Steps>
<Step title="Query Knowledge">
```rust
use praisonai::{Agent, Knowledge};

let knowledge = Knowledge::new()
    .add_file("docs/")
    .build()?;

let agent = Agent::new()
    .name("Assistant")
    .knowledge(knowledge)
    .build()?;

// Query returns relevant information
agent.chat("What is the refund policy?").await?;
```
</Step>
