## Quick Start

<Steps>
<Step title="Simple Loop">
```rust
use praisonai::loop_until;

let result = loop_until(
    agent,
    "Process items",
    |output| output.contains("complete")
).await?;
```
</Step>

<Step title="Counted Loop">
```rust
use praisonai::loop_n;

// Run exactly 5 iterations
let results = loop_n(agent, "Generate idea", 5).await?;

for (i, result) in results.iter().enumerate() {
    println!("Iteration {}: {}", i + 1, result);
}
```
</Step>
</Steps>

---

## Loop Types

| Function | Description |
|----------|-------------|
| `loop_n(agent, task, n)` | Run N times |
| `loop_until(agent, task, condition)` | Run until condition met |
| `loop_while(agent, task, condition)` | Run while condition true |

---

## Configuration Options

```rust
use praisonai::loops::LoopConfig;

let config = LoopConfig::new()
    .max_iterations(20)
    .delay_ms(100)
    .break_on_error(true);
```

| Option | Type | Default | Description |
|--------|------|---------|-------------|
| `max_iterations` | `usize` | `100` | Safety limit |
| `delay_ms` | `u64` | `0` | Delay between iterations |
| `break_on_error` | `bool` | `false` | Stop on error |

## Best Practices

<AccordionGroup>
  <Accordion title="Always set iteration limits">
    Use `max_iterations` to prevent infinite loops.
  </Accordion>
  
  <Accordion title="Use clear exit conditions">
    Make termination conditions explicit and testable.
  </Accordion>
</AccordionGroup>
