## Quick Start

<Steps>
<Step title="Configure Failover">
```rust
use praisonai::Agent;

let agent = Agent::new()
    .name("Reliable Bot")
    .model("gpt-4o")  // Primary
    .fallback_model("claude-3-sonnet")  // Backup
    .fallback_model("ollama/llama3")  // Last resort
    .build()?;

// Automatically fails over if API is down
```
</Step>
</Steps>

---

## When Failover Triggers

| Trigger | Action |
|---------|--------|
| API timeout | Try next provider |
| Rate limit | Try next provider |
| Service down | Try next provider |
| All fail | Return error |

---

## Best Practices

<AccordionGroup>
  <Accordion title="Order by preference">
    Put best provider first, cheapest backup last.
  </Accordion>
  
  <Accordion title="Include local fallback">
    Add Ollama as last resort for offline resilience.
  </Accordion>
</AccordionGroup>
