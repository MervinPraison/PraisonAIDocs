# Skills

The Skills module supports the open Agent Skills standard (agentskills.io), enabling agents to load and use modular capabilities through SKILL.md files.

## Quick Start

```rust
use praisonai::skills::{SkillLoader, SkillProperties};

// Load a skill
let mut loader = SkillLoader::new();
let metadata = loader.load_metadata(Path::new("./skills/research/SKILL.md"))?;

println!("Skill: {}", metadata.name);
println!("Description: {}", metadata.description);

// Full skill loading
let skill = loader.load_full(&metadata.name)?;
println!("Tools: {:?}", skill.get_allowed_tools());
```

## SkillProperties

Properties parsed from SKILL.md frontmatter:

```rust
use praisonai::skills::SkillProperties;

let props = SkillProperties::new("web_research", "Research topics online")
    .with_allowed_tools(vec!["web_search", "read_url"]);

println!("Name: {}", props.name);
println!("Description: {}", props.description);
println!("Tools: {:?}", props.get_allowed_tools());

// Validate
match props.validate() {
    Ok(()) => println!("Valid skill"),
    Err(e) => eprintln!("Invalid: {}", e),
}
```

### Properties Fields

| Field | Type | Description |
|-------|------|-------------|
| `name` | String | Skill name |
| `description` | String | Description |
| `version` | Option | Version string |
| `author` | Option | Author name |
| `allowed_tools` | Option | Allowed tools |
| `requires` | Option | Dependencies |
| `compatibility` | Option | SDK compatibility |

## SkillMetadata

Lightweight metadata for skill discovery:

```rust
use praisonai::skills::{SkillMetadata, SkillProperties};

let props = SkillProperties::new("analysis", "Data analysis skill");
let metadata = SkillMetadata::from_properties(&props)?;

println!("Name: {}", metadata.name);
println!("Description: {}", metadata.description);
println!("Path: {:?}", metadata.path);
```

## SkillLoader

Progressive skill loading:

```rust
use praisonai::skills::SkillLoader;
use std::path::Path;

let mut loader = SkillLoader::new();

// Level 1: Load metadata only (lightweight)
let metadata = loader.load_metadata(Path::new("./skills/coding/SKILL.md"))?;
println!("Found: {} - {}", metadata.name, metadata.description);

// Level 2: Load full skill when needed
let skill = loader.load_full(&metadata.name)?;
println!("Instructions loaded");

// Stats
println!("Skills loaded: {}", loader.skill_count());
println!("Metadata loaded: {}", loader.metadata_count());
```

### Loader Methods

| Method | Description |
|--------|-------------|
| `load_metadata(path)` | Load only metadata |
| `load_full(name)` | Load full skill |
| `parse_frontmatter(content)` | Parse YAML frontmatter |
| `parse_body(content)` | Parse markdown body |
| `skill_count()` | Count loaded skills |
| `metadata_count()` | Count metadata entries |

## SKILL.md Format

Skills are defined in SKILL.md files:

```markdown
---
name: web_research
description: Research topics using web search
version: 1.0.0
author: Your Name
allowed_tools:
  - web_search
  - read_url
  - summarize
requires:
  - internet_access
compatibility: ">=0.5.0"
---

# Web Research Skill

This skill enables comprehensive web research.

## Instructions

1. Start by searching for the topic
2. Read relevant pages
3. Summarize findings

## Examples

Query: "Latest AI trends"
Action: Search, read top results, synthesize
```

## SkillRegistry

Manage multiple skills:

```rust
use praisonai::skills::{SkillRegistry, SkillLoader};
use std::path::Path;

let mut registry = SkillRegistry::new();

// Discover skills in directory
registry.discover(Path::new("./skills"))?;

// List available skills
for skill in registry.list() {
    println!("- {} ({})", skill.name, skill.description);
}

// Get a specific skill
if let Some(skill) = registry.get("web_research") {
    println!("Found: {}", skill.name);
}

// Load on demand
let full_skill = registry.load("web_research")?;
```

## Example: Skill-Enabled Agent

```rust
use praisonai::{Agent, Result};
use praisonai::skills::{SkillLoader, SkillRegistry};
use std::path::Path;

#[tokio::main]
async fn main() -> Result<()> {
    // Discover skills
    let mut registry = SkillRegistry::new();
    registry.discover(Path::new("./skills"))?;
    
    println!("Available skills:");
    for skill in registry.list() {
        println!("  - {}: {}", skill.name, skill.description);
    }
    
    // Load a skill
    let research_skill = registry.load("web_research")?;
    
    // Create agent with skill instructions
    let agent = Agent::new()
        .name("skilled_agent")
        .instructions(&research_skill.instructions.unwrap_or_default())
        .tools(research_skill.get_allowed_tools())
        .build()?;
    
    // Use the skill
    let result = agent.chat("Research the latest AI developments").await?;
    println!("{}", result);
    
    Ok(())
}
```
