## Quick Start

<Steps>
<Step title="Enable Reasoning">
```rust
use praisonai::Agent;

let agent = Agent::new()
    .name("Thinker")
    .reasoning(true)
    .build()?;

agent.chat("What causes rain?").await?;
// Shows step-by-step reasoning
```
</Step>
</Steps>
