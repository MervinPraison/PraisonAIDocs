## Quick Start

<Steps>
<Step title="Embed in Application">
```rust
use praisonai::Agent;

// Create agent as part of your application
let agent = Agent::new()
    .name("Assistant")
    .build()?;

// Use in your app logic
async fn handle_user_query(query: &str) -> String {
    agent.chat(query).await.unwrap()
}
```
</Step>
</Steps>
