## Usage

```python
cond = DictCondition(
        {"approved": ["publish"], "rejected": ["revise"]},
        key="decision"
    )
    result = cond.evaluate({"decision": "approved"})  # True
    targets = cond.get_target({"decision": "approved"})  # ["publish"]
```
