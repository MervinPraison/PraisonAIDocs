## Usage

```python
# Enable with defaults
    agent = Agent(instructions="...", context=True)
    
    # Custom configuration
    agent = Agent(
        instructions="...",
        context=ContextConfig(
            auto_compact=True,
            session_tracking=True,      # Track goal/plan/progress
            aggregate_memory=True,       # Concurrent multi-memory fetch
        )
    )
```
