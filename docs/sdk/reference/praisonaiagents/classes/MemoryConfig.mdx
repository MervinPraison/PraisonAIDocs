## Usage

```python
    Agent(memory=True)
    
    # With backend
    Agent(memory=MemoryConfig(backend="redis"))
    
    # Full config with learning
    Agent(memory=MemoryConfig(
        backend="sqlite",
        user_id="user123",
        session_id="session456",
        auto_memory=True,
        learn=True,  # Enable continuous learning
    ))
    
    # With detailed learn config
    Agent(memory=MemoryConfig(
        learn=LearnConfig(
            persona=True,
            insights=True,
            patterns=True,
        )
    ))
```
