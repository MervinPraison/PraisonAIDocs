# WebConfig

*Module: [feature_configs](../modules/feature_configs)*

Configuration for web search and fetch capabilities.

Consolidates: web_search, web_fetch

Usage:
    # Simple enable
    Agent(web=True)
    
    # With config
    Agent(web=WebConfig(
        search=True,
        fetch=True,
        search_provider="duckduckgo",
        max_results=5,
    ))
