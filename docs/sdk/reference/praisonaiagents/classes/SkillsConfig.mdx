## Usage

```python
# Simple list
    Agent(skills=["./my-skill", "code-review"])
    
    # With config
    Agent(skills=SkillsConfig(
        paths=["./my-skill"],
        dirs=["~/.praisonai/skills/"],
        auto_discover=True,
    ))
```
