# SkillsConfig

*Module: [feature_configs](../modules/feature_configs)*

Configuration for agent skills.

Consolidates: skills, skills_dirs

Usage:
    # Simple list
    Agent(skills=["./my-skill", "code-review"])
    
    # With config
    Agent(skills=SkillsConfig(
        paths=["./my-skill"],
        dirs=["~/.praison/skills/"],
        auto_discover=True,
    ))

## Properties

| Property | Type |

| `dirs` | `List` |
| `auto_discover` | `bool` |
