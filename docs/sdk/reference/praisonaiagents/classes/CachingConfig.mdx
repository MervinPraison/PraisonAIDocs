# CachingConfig

*Module: [feature_configs](../modules/feature_configs)*

Configuration for caching behavior.

Consolidates: cache, prompt_caching

Usage:
    # Simple enable
    Agent(caching=True)
    
    # With config
    Agent(caching=CachingConfig(
        enabled=True,
        prompt_caching=True,
    ))

## Properties

| Property | Type |
|----------|------|
| `enabled` | `bool` |
| `prompt_caching` | `Optional` |
