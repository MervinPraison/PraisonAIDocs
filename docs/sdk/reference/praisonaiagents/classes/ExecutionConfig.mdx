# ExecutionConfig

*Module: [feature_configs](../modules/feature_configs)*

Configuration for agent execution limits.

Consolidates: max_iter, max_rpm, max_execution_time, max_retry_limit

Usage:
    # Simple preset
    Agent(execution="thorough")
    
    # With config
    Agent(execution=ExecutionConfig(
        max_iter=50,
        max_rpm=100,
        max_execution_time=300,
        max_retry_limit=5,
    ))

## Properties

| Property | Type |
|----------|------|
| `max_iter` | `int` |
| `max_rpm` | `Optional` |
| `max_execution_time` | `Optional` |
| `max_retry_limit` | `int` |
