# ExpressionCondition

> Defined in the [**evaluator**](../modules/evaluator) module.

<Badge color="blue">AI Agent</Badge>

String-based condition evaluator for expressions like "&#123;&#123;score&#125;&#125; &gt; 80".

Supports:
    - Numeric comparison: "&#123;&#123;var&#125;&#125; &gt; 80", "&#123;&#123;var&#125;&#125; &gt;= 50", "&#123;&#123;var&#125;&#125; &lt; 10"
    - String equality: "&#123;&#123;var&#125;&#125; == approved", "&#123;&#123;var&#125;&#125; != rejected"
    - Contains check: "error in &#123;&#123;message&#125;&#125;", "&#123;&#123;status&#125;&#125; contains success"
    - Boolean: "&#123;&#123;flag&#125;&#125;" (truthy check)
    - Nested property: "&#123;&#123;item.score&#125;&#125; &gt;= 60"
