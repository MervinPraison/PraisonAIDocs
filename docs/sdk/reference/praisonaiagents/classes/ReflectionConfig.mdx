# ReflectionConfig

*Module: [feature_configs](../modules/feature_configs)*

Configuration for self-reflection.

Consolidates: self_reflect, max_reflect, min_reflect, reflect_llm, reflect_prompt

Usage:
    # Simple enable
    Agent(reflection=True)
    
    # With config
    Agent(reflection=ReflectionConfig(
        min_iterations=1,
        max_iterations=3,
        llm="gpt-4o",
        prompt="Evaluate your response for accuracy...",
    ))

## Properties

| Property | Type |
|----------|------|
| `min_iterations` | `int` |
| `max_iterations` | `int` |
| `llm` | `Optional` |
| `prompt` | `Optional` |
