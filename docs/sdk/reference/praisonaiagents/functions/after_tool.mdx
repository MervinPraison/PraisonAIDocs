# after_tool

*Module: [middleware](../modules/middleware)*

Decorator to mark a function as an after_tool hook.

The function receives a ToolResponse and should return a (possibly modified) ToolResponse.

Example:
    @after_tool
    def log_result(response):
        print(f"Tool &#123;response.tool_name&#125; returned: &#123;response.result&#125;")
        return response

## Signature

