# aembedding

*Module: [embed](../modules/embed)*

Async: Generate embeddings for text using LiteLLM.

This is the async version of embedding() for use in async contexts.
See embedding() for full documentation.

Example:
    &gt;&gt;&gt; from praisonaiagents import aembedding
    &gt;&gt;&gt; result = await aembedding("Hello, world!")
    &gt;&gt;&gt; print(len(result.embeddings[0]))
    1536
