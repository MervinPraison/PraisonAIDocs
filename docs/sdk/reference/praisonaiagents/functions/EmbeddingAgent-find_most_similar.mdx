<ResponseField name="Returns" type="List[Dict[str, Any]]">

</ResponseField>

## Usage

```python
agent = EmbeddingAgent()
    results = agent.find_most_similar(
        "What is AI?",
        ["Artificial intelligence is...", "Machine learning...", "Deep learning..."],
        top_k=2
    )
    for r in results:
        print(f"{r['score']:.2f}: {r['text'][:50]}...")
```
