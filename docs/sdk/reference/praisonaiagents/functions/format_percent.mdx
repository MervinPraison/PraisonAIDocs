# format_percent

<div className="flex items-center gap-2">
  <Badge color="teal">Function</Badge>
</div>

> This function is defined in the [**context**](../modules/context) module.

Smart percentage formatting for context utilization display.

- For values &lt; 0.1%: shows "&lt;0.1%"
- For values &lt; 1%: shows 2 decimal places (e.g., "0.02%")
- For values &gt;= 1%: shows 1 decimal place (e.g., "5.3%")

## Signature

```python
def format_percent(value: float) -> str
```

## Parameters

<ParamField query="value" type="float" required={true}>
  A ratio (0.0 to 1.0+), NOT already multiplied by 100
</ParamField>

### Returns

<ResponseField name="Returns" type="str">
  Formatted percentage string
</ResponseField>
