## Classes

### Chunking

A unified class for text chunking with various chunking strategies.

<Accordion title="Constructor Parameters">

| Parameter | Type | Required | Default |
|-----------|------|----------|---------|
| `chunker_type` | `str` | No | `'token'` |
| `chunk_size` | `int` | No | `512` |
| `chunk_overlap` | `int` | No | `128` |
| `tokenizer_or_token_counter` | `str` | No | `'gpt2'` |
| `embedding_model` | `Optional` | No | `None` |
</Accordion>

<Accordion title="Methods">
- **SUPPORTED_CHUNKERS**(``) → `Dict`
  Lazy load chunker classes.
- **embedding_model**(``) → `Any`
  Lazy load the embedding model.
- **chunker**(``) → `Any`
  Lazy load the chunker instance.
- **chunk**(`text: Union`) → `Union`
  Chunk text using the configured chunking strategy.
</Accordion>
