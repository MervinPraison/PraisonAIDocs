# AutoGenerator

*Module: [auto](../modules/auto)*

Auto-generates agents.yaml files from a topic description.

Inherits from BaseAutoGenerator for shared LLM client functionality.

Usage:
    generator = AutoGenerator(framework="crewai", topic="Create a movie script")
    path = generator.generate()

## Constructor

| Parameter | Type | Required | Default |
|-----------|------|----------|---------|
| `topic` | `Any` | No | `'Movie Story writing about AI'` |
| `agent_file` | `Any` | No | `'test.yaml'` |
| `framework` | `Any` | No | `'crewai'` |
| `config_list` | `Optional` | No | `None` |
| `pattern` | `str` | No | `'sequential'` |
| `single_agent` | `bool` | No | `False` |
