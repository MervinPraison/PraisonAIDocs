# ChatConfig

*Module: [chat](../modules/chat)*

Configuration for the PraisonAI Chat server.

## Constructor

| Parameter | Type | Required | Default |
|-----------|------|----------|---------|
| `host` | `str` | No | `'0.0.0.0'` |
| `port` | `int` | No | `8000` |
| `debug` | `bool` | No | `False` |
| `auth_enabled` | `bool` | No | `False` |
| `session_id` | `Optional` | No | `None` |
