# StreamingTracker

*Module: [profiler](../modules/profiler)*

Track streaming operations (LLM responses).

Usage:
    tracker = StreamingTracker("chat")
    tracker.start()
    tracker.first_token()  # Mark TTFT
    for chunk in stream:
        tracker.chunk()
    tracker.end(total_tokens=100)

## Constructor

| Parameter | Type | Required | Default |
|-----------|------|----------|---------|
| `name` | `str` | Yes | `-` |
