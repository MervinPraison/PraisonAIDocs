# ScheduleParser.parse

*Class: [ScheduleParser](../classes/ScheduleParser)*

Parse schedule expression and return interval in seconds.

Supported formats:
- "daily" -&gt; 86400 seconds
- "hourly" -&gt; 3600 seconds
- "*/30m" -&gt; 1800 seconds (every 30 minutes)
- "*/1h" -&gt; 3600 seconds (every 1 hour)
- "60" -&gt; 60 seconds (plain number)

Args:
    schedule_expr: Schedule expression string
    
Returns:
    Interval in seconds
    
Raises:
    ValueError: If schedule format is not supported

## Signature

```python
def parse(schedule_expr: str) -> int
```
