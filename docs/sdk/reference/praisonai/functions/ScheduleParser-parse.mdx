> This is a method of the [**ScheduleParser**](../classes/ScheduleParser) class in the [**agent_scheduler**](../modules/agent_scheduler) module.

Parse schedule expression and return interval in seconds.

Supported formats:
- "daily" -&gt; 86400 seconds
- "hourly" -&gt; 3600 seconds
- "*/30m" -&gt; 1800 seconds (every 30 minutes)
- "*/1h" -&gt; 3600 seconds (every 1 hour)
- "60" -&gt; 60 seconds (plain number)
