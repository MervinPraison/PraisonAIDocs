# praisonai.tools

<Badge color="orange">Rust SDK</Badge>

Tool system for PraisonAI

This module provides the tool abstraction for agents.
Tools are functions that agents can call to perform actions.

# Example

```rust,ignore
use praisonai_core::&#123;tool, Tool&#125;;

#[tool(description = "Search the web")]
async fn search(query: String) -&gt; String &#123;
format!("Results for: &#123;&#125;", query)
&#125;
```
