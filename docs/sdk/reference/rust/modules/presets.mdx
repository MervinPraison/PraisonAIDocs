# presets

<Badge color="orange">Rust AI Agent SDK</Badge>

Preset configurations for PraisonAI

This module provides preset configurations that match the Python SDK's presets.py.
Presets allow users to configure features using simple string names.

# Example

```rust,ignore
use praisonai::presets::&#123;MEMORY_PRESETS, resolve_memory_preset&#125;;

// Get preset by name
if let Some(config) = resolve_memory_preset("sqlite") &#123;
println!("Backend: &#123;&#125;", config.backend);
&#125;
```
