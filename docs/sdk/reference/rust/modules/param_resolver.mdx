# param_resolver

<Badge color="orange">Rust AI Agent SDK</Badge>

Parameter Resolver

Unified parameter resolution following precedence rules:
Instance &gt; Config &gt; Dict &gt; Array &gt; String &gt; Bool &gt; Default

## Import

```rust
use praisonai::parity::param_resolver::*;
```
