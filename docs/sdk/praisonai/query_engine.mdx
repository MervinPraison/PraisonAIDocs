# Query Engine Module

The Query Engine module provides configurable query processing modes for answer synthesis.

## Import

```python
from praisonai.knowledge import QueryEngine, get_query_engine
```

## Quick Example

```python
from praisonai.knowledge import get_query_engine

# Get a query engine
engine = get_query_engine(mode="sub_question")

# Query the knowledge base
response = engine.query("Compare Python and Java")
print(response.answer)
```

## Features

- Default mode for direct retrieval and synthesis
- Sub-question decomposition for complex queries
- Summarization mode for overview generation
- SQL mode for structured data queries
- Router mode for automatic mode selection

## CLI Usage

```bash
--query-mode <mode>
# or
--mode <mode>
```

## Available Modes

| Mode | Description |
|------|-------------|
| `default` | Direct retrieval + answer synthesis |
| `sub_question` | Decompose into sub-questions |
| `summarize` | Summarize all retrieved content |
| `sql` | Generate SQL queries (structured data) |
| `router` | Auto-select best mode |

## Usage

### Default Mode

```bash
# Simple retrieve and answer
praisonai knowledge query "What is Python?" --query-mode default
```

### Sub-Question Mode

```bash
# Decompose complex questions
praisonai knowledge query "Compare Python and Java for web development" \
  --query-mode sub_question
```

### Summarize Mode

```bash
# Summarize all relevant content
praisonai knowledge query "Summarize the documentation" --query-mode summarize
```

### Router Mode

```bash
# Auto-select best mode
praisonai knowledge query "What are the pros and cons?" --query-mode router
```

## Mode Selection Guide

| Query Type | Recommended Mode |
|------------|------------------|
| Simple factual | `default` |
| Multi-part questions | `sub_question` |
| Overview requests | `summarize` |
| Unknown complexity | `router` |

## Examples

### Complex Comparison Query

```bash
# Decompose into sub-questions
praisonai knowledge query \
  "What are the advantages and disadvantages of microservices vs monolith?" \
  --query-mode sub_question
```

### Document Summary

```bash
# Get overview of all content
praisonai knowledge query "Give me an overview of the project" \
  --query-mode summarize
```

### Full Pipeline

```bash
# Complete RAG configuration
praisonai knowledge query "How should I architect this system?" \
  --vector-store chroma \
  --retrieval fusion \
  --reranker simple \
  --index-type hybrid \
  --query-mode sub_question
```
