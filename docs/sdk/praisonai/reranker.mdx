# Reranker Module

The Reranker module provides document reranking capabilities to improve search result relevance.

## Import

```python
from praisonai.knowledge import Reranker, get_reranker
```

## Quick Example

```python
from praisonai.knowledge import get_reranker

# Get a reranker
reranker = get_reranker("simple")

# Rerank documents
reranked = reranker.rerank(query="Python basics", documents=docs)
```

## Features

- Multiple reranking strategies (simple, LLM, cross-encoder, Cohere)
- Keyword-based scoring
- LLM-based relevance scoring
- Cross-encoder neural reranking
- Cohere Rerank API integration

## CLI Usage

```bash
--reranker <type>
```

## Available Rerankers

| Type | Description | Requirements |
|------|-------------|--------------|
| `none` | No reranking | None |
| `simple` | Keyword-based scoring | None (built-in) |
| `llm` | LLM-based relevance | OpenAI/Anthropic API key |
| `cross_encoder` | Cross-encoder model | `sentence-transformers` |
| `cohere` | Cohere Rerank API | `COHERE_API_KEY` |

## Usage

### No Reranking

```bash
# Skip reranking step
praisonai knowledge query "What is Python?" --reranker none
```

### Simple Reranker (Built-in)

```bash
# Keyword-based, no external deps
praisonai knowledge query "What is Python?" --reranker simple
```

### LLM Reranker

```bash
# Uses LLM to score relevance
export OPENAI_API_KEY=sk-xxx
praisonai knowledge query "What is Python?" --reranker llm
```

### Cross-Encoder Reranker

```bash
# Requires sentence-transformers
pip install sentence-transformers
praisonai knowledge query "What is Python?" --reranker cross_encoder
```

### Cohere Reranker

```bash
# Requires Cohere API key
export COHERE_API_KEY=xxx
praisonai knowledge query "What is Python?" --reranker cohere
```

## Reranker Selection Guide

| Use Case | Recommended |
|----------|-------------|
| Fast, no dependencies | `simple` |
| Best accuracy | `cohere` or `cross_encoder` |
| Already using LLM | `llm` |
| Minimal latency | `none` |

## Examples

### Combining with Retrieval

```bash
# Fusion retrieval + simple reranking
praisonai knowledge query "Compare approaches" \
  --retrieval fusion \
  --reranker simple
```

### Full Pipeline

```bash
# Complete RAG with reranking
praisonai knowledge query "Best practices for deployment" \
  --vector-store chroma \
  --retrieval fusion \
  --reranker simple \
  --query-mode default
```
