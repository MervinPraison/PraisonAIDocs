# Vector Store Module

The Vector Store module provides configurable vector storage backends for semantic search and document retrieval.

## Import

```python
from praisonai.knowledge import VectorStore, get_vector_store
```

## Quick Example

```python
from praisonai.knowledge import get_vector_store

# Get a vector store instance
store = get_vector_store("chroma")

# Add documents
store.add_documents(["Document 1", "Document 2"])

# Query
results = store.query("search query", top_k=5)
```

## Features

- Multiple backend support (ChromaDB, Pinecone, Qdrant, Weaviate)
- In-memory store for testing
- Persistent local storage
- Cloud vector database integration
- Automatic embedding generation

## CLI Usage

```bash
--vector-store <name>
```

## Available Stores

| Name | Description | Requirements |
|------|-------------|--------------|
| `memory` | In-memory store | None (built-in) |
| `chroma` | ChromaDB | `pip install chromadb` |
| `pinecone` | Pinecone | `pip install pinecone-client` |
| `qdrant` | Qdrant | `pip install qdrant-client` |
| `weaviate` | Weaviate | `pip install weaviate-client` |

## Usage

### With Query Command

```bash
# Use in-memory store
praisonai knowledge query "What is Python?" --vector-store memory

# Use ChromaDB (default)
praisonai knowledge query "What is Python?" --vector-store chroma

# Use Pinecone
praisonai knowledge query "What is Python?" --vector-store pinecone

# Use Qdrant
praisonai knowledge query "What is Python?" --vector-store qdrant
```

### With Add Command

```bash
# Add to specific store
praisonai knowledge add document.pdf --vector-store chroma
```

## Environment Variables

Some stores require API keys:

```bash
# Pinecone
export PINECONE_API_KEY=your-key
export PINECONE_ENVIRONMENT=us-east-1

# Qdrant Cloud
export QDRANT_API_KEY=your-key
export QDRANT_URL=https://your-cluster.qdrant.io
```

## Examples

### In-Memory Store (Testing)

```bash
# Quick testing without persistence
praisonai knowledge add test.txt --vector-store memory
praisonai knowledge query "test query" --vector-store memory
```

### ChromaDB (Local Persistence)

```bash
# Default - persists to .praison/knowledge/
praisonai knowledge add docs/ --vector-store chroma
praisonai knowledge query "How to authenticate?"
```

### Cloud Stores

```bash
# Pinecone (requires API key)
export PINECONE_API_KEY=pk-xxx
praisonai knowledge query "query" --vector-store pinecone

# Qdrant Cloud
export QDRANT_API_KEY=xxx
praisonai knowledge query "query" --vector-store qdrant
```
