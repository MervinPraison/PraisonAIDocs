# Module praisonaiagents

The `praisonaiagents` package provides a lightweight, high-performance framework for building AI agents with support for multi-agent orchestration, memory, knowledge bases, and tool integration.

## Quick Start

```python
from praisonaiagents import Agent

agent = Agent(instructions="You are a helpful assistant")
response = agent.start("Hello!")
print(response)
```

## Public Exports

The following classes and functions are exported from the package root:

```python
from praisonaiagents import (
    # Core
    Agent, Agents, Task, Tools,
    # Agent variants
    ImageAgent, ContextAgent, DeepResearchAgent,
    QueryRewriterAgent, PromptExpanderAgent,
    # Multi-agent
    AutoAgents, AutoRagAgent,
    # Data
    Session, Memory, Knowledge, db, obs,
    # Workflows
    Workflow, Pipeline, Route, Parallel, Loop, Repeat,
    # Tools
    MCP, tool, BaseTool, ToolResult,
    # Handoff
    Handoff, handoff,
    # Guardrails
    GuardrailResult, LLMGuardrail,
    # Display
    display_interaction, display_error, display_tool_call,
    # Telemetry
    get_telemetry, enable_telemetry, disable_telemetry,
)
```

## Installation

```bash
# Core package
pip install praisonaiagents

# With memory support
pip install "praisonaiagents[memory]"

# With knowledge/RAG support  
pip install "praisonaiagents[knowledge]"

# With MCP support
pip install "praisonaiagents[mcp]"
```
