# UI Module

The UI module exposes protocol adapters for agent frontends and inter-agent communication.

## Installation

```bash
pip install praisonaiagents
# Optional A2UI (Google a2ui-agent-sdk):
pip install praisonaiagents[a2ui]
```

## Components

### AGUI

AG-UI protocol for CopilotKit-compatible SSE streaming.

```python
from praisonaiagents import Agent, AGUI
from fastapi import FastAPI

agent = Agent(name="assistant")
agui = AGUI(agent=agent)

app = FastAPI()
app.include_router(agui.get_router())
```

See [AG-UI server](/docs/deploy/servers/agui).

### A2A

Agent-to-Agent JSON-RPC protocol.

```python
from praisonaiagents import Agent, A2A

agent = Agent(name="assistant")
a2a = A2A(agent=agent, url="http://localhost:8000/a2a")
app.include_router(a2a.get_router())
```

Sync `message/send` can emit A2UI `DataPart` artifacts when the agent used `send_a2ui_messages`. See [A2A Protocol](/docs/features/a2a).

### A2UI

Optional declarative UI via Google's A2UI SDK (thin adapter in core).

```python
from praisonaiagents import Agent, A2UI
from praisonaiagents.tools.a2ui_tools import send_a2ui_messages

agent = Agent(name="assistant", tools=[send_a2ui_messages])
part = A2UI.create_part({"createSurface": {"surfaceId": "main", "catalogId": "basic"}})
```

Tool output contract:

```python
from praisonaiagents.ui.protocols import A2UI_MIME_TYPE, A2UIToolResultProtocol
```

For rendering and parsing, see [Integrate A2UI with Your Frontend](/docs/features/integrate-a2ui-frontend).
