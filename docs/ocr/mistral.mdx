## Usage

```python
from praisonaiagents import OCRAgent

agent = OCRAgent()
text = agent.read("document.pdf")
print(text)
```

## From Image

```python
text = agent.read("screenshot.png")
```

## With Options

```python
result = agent.extract(
    "document.pdf",
    pages=[1, 2, 3],
    include_image_base64=True
)
for page in result.pages:
    print(page.markdown)
```
