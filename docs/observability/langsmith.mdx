# LangSmith Integration

[LangSmith](https://smith.langchain.com/) is LangChain's platform for debugging, testing, and monitoring LLM applications.

## Setup

### 1. Install Dependencies

```bash
pip install opentelemetry-sdk opentelemetry-exporter-otlp
```

### 2. Set Environment Variables

```bash
export LANGSMITH_API_KEY=lsv2_xxx
export LANGSMITH_PROJECT=my-project  # optional
```

### 3. Initialize

```python
from praisonai_tools.observability import obs

obs.init(provider="langsmith")
```

## Usage

```python
from praisonai_tools.observability import obs
from praisonaiagents import Agent

obs.init(provider="langsmith")

agent = Agent(
    instructions="You are a helpful assistant.",
    model="gpt-4o-mini",
)

with obs.trace("chat-session"):
    response = agent.chat("What is AI?")
    print(response)
```

## Configuration Options

| Option | Environment Variable | Description |
|--------|---------------------|-------------|
| `api_key` | `LANGSMITH_API_KEY` | Your LangSmith API key |
| `project` | `LANGSMITH_PROJECT` | Project name (default: "default") |
| `endpoint` | `LANGSMITH_ENDPOINT` | API endpoint |
