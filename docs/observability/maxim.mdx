# Maxim Integration

[Maxim](https://getmaxim.ai/) provides LLM observability and logging.

## Setup

```bash
pip install maxim-py
export MAXIM_API_KEY=xxx
```

## Usage

```python
from praisonai_tools.observability import obs

obs.init(provider="maxim")

with obs.trace("workflow"):
    # Your agent code
    pass
```
