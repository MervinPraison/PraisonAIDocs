# Weave Integration

[Weave](https://weave-docs.wandb.ai/) by Weights & Biases provides LLM tracing.

## Setup

```bash
pip install weave
export WANDB_API_KEY=xxx
```

## Usage

```python
from praisonai_tools.observability import obs

obs.init(provider="weave")

with obs.trace("workflow"):
    # Your agent code
    pass
```
