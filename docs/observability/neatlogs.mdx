# Neatlogs Integration

Neatlogs provides LLM logging and observability.

## Setup

```bash
export NEATLOGS_API_KEY=xxx
```

## Usage

```python
from praisonai_tools.observability import obs

obs.init(provider="neatlogs")

with obs.trace("workflow"):
    # Your agent code
    pass
```
