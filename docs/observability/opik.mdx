# Opik Integration

[Opik](https://www.comet.com/docs/opik/) by Comet provides LLM evaluation and tracing.

## Setup

```bash
pip install opik
export OPIK_API_KEY=xxx
```

## Usage

```python
from praisonai_tools.observability import obs

obs.init(provider="opik")

with obs.trace("workflow"):
    # Your agent code
    pass
```
