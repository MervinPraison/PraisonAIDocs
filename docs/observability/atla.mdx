# Atla Integration

[Atla](https://app.atla-ai.com/) provides LLM observability.

## Setup

```bash
pip install atla-insights
export ATLA_API_KEY=xxx
```

## Usage

```python
from praisonai_tools.observability import obs

obs.init(provider="atla")

with obs.trace("workflow"):
    # Your agent code
    pass
```
