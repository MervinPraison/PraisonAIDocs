# LangDB Integration

LangDB provides SQL-based LLM observability.

## Setup

```bash
export LANGDB_API_KEY=xxx
```

## Usage

```python
from praisonai_tools.observability import obs

obs.init(provider="langdb")

with obs.trace("workflow"):
    # Your agent code
    pass
```
